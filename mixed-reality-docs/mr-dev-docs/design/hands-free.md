## <a name="transitioning-in-and-out-of-hands-free"></a><span data-ttu-id="46ed2-125">Перевод в руки и выход из них</span><span class="sxs-lookup"><span data-stu-id="46ed2-125">Transitioning in and out of hands-free</span></span>

<span data-ttu-id="46ed2-127">Если требуется, чтобы приложение всегда использовало практически бесплатно, с помощью вдаваясь, настраиваемых речевых команд или одной команды Voice, выберите "Select", а затем убедитесь, что в пользовательском интерфейсе нужны соответствующие места.</span><span class="sxs-lookup"><span data-stu-id="46ed2-127">If the requirement of the application is that it will always be used hands-free, whether by using dwell, custom voice commands, or the single voice command, "select", then make sure to make the appropriate accommodations in your UI.</span></span>
