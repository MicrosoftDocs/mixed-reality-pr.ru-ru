# <a name="mixed-reality-openxr-plugin-release-notes"></a>Заметки о выпуске подключаемого модуля OpenXR для смешанной реальности

## <a name="111---2021-10-15"></a>1.1.1 — 15.10.2021

* Исправлена ошибка, из-за которой не удавалось выполнить сборку проектов, если проект также ссылался на пакет DotNetWinRT.

## <a name="110---2021-10-07"></a>1.1.0 — 07.10.2021

* Добавлены новые API для пакета перемещения пространственной привязки: Microsoft.MixedReality.OpenXR.XRAnchorTransferBatch.
* Включена поддержка XRMeshSubsystem через расширения распознавания сцен OpenXR.
* Включена поддержка среды выполнения для удаленного взаимодействия OpenXR 2.7 с сопоставлением Spatial Anchor Store и Surface в приложениях голографического удаленного взаимодействия.
* Удалена прямая зависимость пакета для пакета ARSubsystem.  Теперь она предоставляется неявно через пакет ARFoundation.
* Исправлена ошибка, из-за которой пользовательский интерфейс Unity на короткое время зависал, когда компонент голографического удаленного взаимодействия не мог подключиться к удаленному приложению проигрывателя.
* Исправлена ошибка, из-за которой использование сохраненной привязки могло приводить к ошибке "Элемент с тем же ключом уже добавлен".
* Включена поддержка утвержденного расширения XR_MSFT_scene_understanding вместо версии _preview3.
* Исправлена ошибка, из-за которой не удавалось выполнить сборку проектов с установленным подключаемым модулем Windows XR Plugin и включенным удаленным взаимодействием через приложение.

## <a name="103---2021-09-07"></a>1.0.3 — 07.09.2021

* Включена поддержка расширения MSFT для сохранения пространственных привязок OpenXR.
* Исправлена ошибка, из-за которой параметры удаленного взаимодействия редактора находились в PackageSettings вместо UserSettings.
* Исправлена ошибка, из-за которой некоторые привязки могли не сохраняться после очистки XRAnchorStore.
* Исправлена ошибка, из-за которой при переключении между сценами Unity создавались дополнительные привязки.

## <a name="102---2021-08-05"></a>1.0.2 — 05.08.2021

* Добавлена зависимость от подключаемого модуля Unity OpenXR 1.2.8.
* Исправлена ошибка, из-за которой объекты ARAnchor иногда не удалялись должным образом.
* Исправлена ошибка, из-за которой иногда сообщалось о недопустимых изменениях ARAnchor после перезапуска компонента голографического удаленного взаимодействия для режима воспроизведения.
* Исправлена ошибка, из-за которой конфигурации представления не передавались должным образом при использовании компонента голографического удаленного взаимодействия для режима воспроизведения.
* Включена более точная проверка параметров с более точными сообщениями при использовании компонента голографического удаленного взаимодействия для режима воспроизведения.
* Включена проверка параметра Initialize XR on Startup (Инициализировать смешанную реальность при запуске) при использовании компонента голографического удаленного взаимодействия для режима воспроизведения.

## <a name="101---2021-07-13"></a>1.0.1 — 13.07.2021

* Зависит от подключаемого модуля Unity OpenXR 1.2.3.
* Обновлена среда выполнения компонента голографического удаленного взаимодействия до версии 2.6.0.
* Удалена группа компонента голографического удаленного взаимодействия для режима воспроизведения из диалогового интерфейса с параметрами Unity; теперь компонент предоставляется независимо.
* Исправлена ошибка, из-за которой процессу сборки не удается найти app.cpp при построении проекта Unity типа XAML.

## <a name="100---2021-06-18"></a>1.0.0 — 18.06.2021

* Исправлена ошибка, из-за которой запуск XRAnchorSubsystem всегда осуществлялся при запуске приложения независимо от наличия ARAnchorManager.
* Исправлена ошибка, из-за которой режим повторного проецирования не работал должным образом.

## <a name="100-preview2---2021-06-14"></a>1.0.0-preview.2 — 2021-06-14

* Добавлена зависимость от подключаемого модуля Unity OpenXR 1.2.2.
* Изменены функции голографического удаленного взаимодействия Holographic Remoting для отдельных групп функций.
* Исправлена ошибка, из-за которой параметр Apply HoloLens 2 project settings (Применить параметры проекта HoloLens 2) изменял цветовое пространство проекта.  Это исправление не требуется для подключаемого модуля Unity OpenXR версии 1.2.0 и выше.
* Исправлена ошибка, при которой входное устройство подключалось без отключения после приостановки и возобновления работы приложения.
* Добавлена поддержка обнаружения подключаемого модуля и текущего состояния отслеживания через ARSession.
* Исправлена ошибка, из-за которой заготовка ARFoundation "AR Default Plane" (Область дополненной реальности по умолчанию) не отображалась.

## <a name="100-preview1---2021-06-02"></a>1.0.0-preview.1 — 2021-06-02

* Поддерживает расширения MSFT для интерпретаций сцен OpenXR вместо расширений для предварительного просмотра.
* Обнаружение плоскости в HoloLens 2 больше не требует предварительных версий сред выполнения OpenXR смешанной реальности.

## <a name="095---2021-05-21"></a>0.9.5 — 2021-05-21

* Добавлена зависимость от подключаемого модуля Unity OpenXR 1.2.0.
* Адаптация к новому пользовательскому интерфейсу компонента (в подключаемом модуле OpenXR 1.2.0) для конфигурации.
* Исправлена ошибка, из-за которой регистрация поставщика обнаруживаемой камеры не отменялась должным образом.
* Исключено несколько дополнительных вариантов использования `[Preserve]`.
* В пользовательском интерфейсе системы ввода обновлено имя контроллера HP Reverb G2 Controller (OpenXR).

## <a name="094---2021-05-20"></a>0.9.4 — 2021-05-20

* Добавлена зависимость от подключаемого модуля Unity OpenXR 1.2.0.
* Добавлен новый API C# для получения модели glTF контроллера движения.
* Добавлен новый API C# для получения включенных конфигураций представлений и задания параметров перепроецирования.
* Добавлен новый API C# для настройки дополнительных параметров для вычисления сеток с помощью XRMeshSubsystem.
* Добавлен новый API C# для подписки на события распознавания жестов и их настройки.
* Добавлено диалоговое окно параметров Windows->XR->Editor Remoting.
* Добавлена поддержка ARM для приложений UWP HoloLens.

## <a name="093---2021-04-29"></a>0.9.3 — 2021-04-29

* Исправлена ошибка, при которой подключение для голографического удаленного взаимодействия было ненадежным.
* Исправлена ошибка, из-за которой производительность отрисовки виртуальной реальности являлась неоптимальной после обновления до подключаемого модуля Unity OpenXR 1.1.1.

## <a name="092---2021-04-21"></a>0.9.2 — 2021-04-21

* Обнаружение плоскости на HoloLens 2 в подключаемом модуле версии 0.9.1 будет работать с версией 105 среды выполнения смешанной реальности OpenXR (предварительная версия).
* Обнаружение плоскости на HoloLens 2 в подключаемом модуле версии 0.9.2 будет работать с версией 106 среды выполнения смешанной реальности OpenXR (предварительная версия).
* Удалены некоторые неиспользуемые обратные вызовы из InputProvider, чтобы предотвратить возврат успешных вызовов с неправильными значениями, таких как XRInputSubsystem.GetTrackingOriginMode (которые не управляются нашей системой ввода).
* Устаревшая версия XRAnchorStore выделена в свой собственный файл, чтобы предотвратить предупреждение консоли Unity.

## <a name="091---2021-04-20"></a>0.9.1 — 2021-04-20

* Добавлена зависимость от подключаемого модуля Unity OpenXR 1.1.1.
* Включена поддержка приложения голографического удаленного взаимодействия (https://aka.ms/openxr-unity-app-remoting) для платформы UWP).
* Исправление UnityException, где XRAnchorStore пытался получить экземпляр параметров за пределами основного потока.

## <a name="090---2021-03-29"></a>0.9.0 — 2021-03-29

* Добавлена поддержка пространственного сопоставления через XRMeshSubsystem и ARMeshManager.
* Добавлен новый API C# для получения дескрипторов OpenXR для поддержки других пакетов Unity, использующих расширения OpenXR.
* Добавлен новый API C# для взаимодействия с API Windows.Perception для поддержки других пакетов Unity, использующих API-интерфейсы WinRT.
* Удалены профили взаимодействия из требуемых функций в наборе функций Windows Mixed Reality, чтобы разработчики могли выбрать контроллеры движений, которые тестировались.
* Добавлена проверка функции редактора голографического удаленного взаимодействия для помощи пользователям в настройке.
* Исправлена ошибка, из-за которой работа редактора Unity аварийно завершалась при выходе из строя режима голографического удаленного взаимодействия после сбоя подключения.
* Исправлена ошибка, при которой неумноженные альфа-текстуры приводили к неоптимальной производительности на HoloLens 2.
* Исправлена ошибка, при которой отслеживание было обнаружено неправильно, если источник сцены был на уровне пола.
* Исправлена ошибка, из-за которой отслеживание сетки исчезало после завершения и загрузки новой сцены.
* Исправлена ошибка, при которой поставщик обнаруживаемой камеры не очищался должным образом.
* Изменено пространство имен API XRAnchorStore на Microsoft.MixedReality.OpenXR.
