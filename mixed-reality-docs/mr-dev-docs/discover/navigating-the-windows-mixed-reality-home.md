# <a name="navigating-the-windows-mixed-reality-home"></a>Навигация по дому с технологией Windows Mixed Reality

Как и при работе с компьютером Windows, Windows Mixed Reality начинается с домашнего компьютера. Домашняя страница Windows Mixed Reality использует нашу присущейную возможность для понимания трехмерных мест и навигации по ним. При использовании HoloLens ваш дом является вашим физическим пространством, но с помощью впечатляющих головок дома это виртуальное место.

Домашняя страница также используется для открытия и размещения приложений и содержимого в меню "Пуск". Вы можете заполнить содержимое смешанной реальности содержимым и многозадачностью, одновременно используя несколько приложений. Те вещи, которые вы поместите на домашнюю страницу, остаются там даже при перезагрузке устройства.

## <a name="start-menu"></a>Меню "Пуск"

![Меню "Пуск" в Microsoft HoloLens](images/start-500px.png)

Меню Пуск состоит из следующих компонентов:
* Сведения о системе (состояние сети, процент аккумулятора, текущее время и том)
* Кортана (на впечатляющих гарнитурах, начальная плитка, на HoloLens в начале)
* Закрепленные приложения
* Кнопка "все приложения" (знак "плюс")
* Кнопки фото и Video для [записи смешанной реальности](../mixed-reality-capture.md)

Переключитесь между представлениями "закрепленные приложения" и "все приложения", нажав кнопки "плюс" или "минус". Чтобы открыть меню "Пуск" в HoloLens, используйте жест раскрытия. На иммерсивного гарнитуре нажмите кнопку Windows на контроллере.

## <a name="launching-apps"></a>Запуск приложений

Чтобы запустить приложение, выберите его при запуске. Меню Пуск исчезнет, и приложение откроется в режиме размещения в виде 2D-окна или [трехмерной модели](../distribute/implementing-3d-app-launchers.md).

Чтобы запустить приложение, необходимо разместить его в вашей домашней папке:
1. Используйте свое [взгляд](../design/gaze-and-commit.md) или контроллер, чтобы разместить нужное приложение в нужном месте. Он автоматически подстраивается (в размере и позиции) в соответствии с местом размещения.
2. Разместите приложение с помощью воздушного касания (HoloLens) или кнопки выбора (иммерсивное гарнитуры). Чтобы отменить действие и вернуть меню Пуск, используйте жест раскрытия или кнопку Windows.

[2D-приложения](../develop/porting-apps/building-2d-apps.md), созданные для настольных систем, мобильных устройств или Xbox, можно изменить для запуска в качестве впечатляющих приложений в смешанной реальности с помощью [API холографикспаце](https://msdn.microsoft.com/library/windows/apps/windows.graphics.holographic.holographicspace.aspx). Иммерсивное приложение выводит пользователя из дома и в иммерсивное взаимодействие. Пользователи могут вернуть домой с помощью жеста раскрытия (HoloLens) или нажатия кнопки Windows на контроллере (иммерсивное гарнитуры).

Приложения также могут запускаться через API приложения для приложения или через Кортану.

![Приложения на домашней странице Windows Mixed Reality](images/mixed-reality-home-500px.png)

## <a name="moving-and-adjusting-apps"></a>Перемещение и настройка приложений

Щелкните **настроить** на панели приложений, чтобы отобразить элементы управления, которые перемещают, масштабируют и поворачивают содержимое смешанной реальности. По завершении нажмите кнопку **Готово**.

![Магазин в режиме коррекции (синий кадр). Примечание. панель приложений (сверху) изменилась для включения кнопок "Готово" и "Удалить".](images/adjust-500px.png)

Различные приложения могут иметь другие параметры на панели приложений. Например, Microsoft ребра имеет варианты *прокрутки*, *перетаскивания* и *масштабирования* . 

![Панель приложений для двумерных приложений, выполняющихся в HoloLens](images/holobar-500px.png)

Кнопка **назад** выполняет переход к ранее просмотренным экранам в приложении. Она останавливается, когда вы достигаете начала работы, показанной в приложении, и не переходите к другим приложениям.

## <a name="getting-around-your-home"></a>Приступая к работе с домашней страницей

С помощью **HoloLens** можно перемещаться по физическому пространству, чтобы перемещаться по дому.

С помощью **впечатляющих головных телефонов** вы можете приступить к работе с плайспаце, чтобы перейти в аналогичную область в виртуальном мире. Для перехода на более длительные расстояния Используйте аналоговый стик на контроллере, чтобы практически «пройти», или можно использовать *телеперенос* для немедленного перехода на более длительные расстояния.

![Телеперенос на домашней странице Windows Mixed Reality](images/teleportation-500px.png)

**Для телепортировать:**
1. Откройте ретиклеи для телетранспорта.
   * Использование [контроллеров движения](../design/motion-controllers.md). Нажмите на аналоговый стик в направлении вперед и удерживайте его в этой должности.
   * Использование контроллера Xbox. Нажмите левый аналоговый стик в направлении вперед и удерживайте его в этой должности.
   * С помощью мыши: удерживайте нажатой правую кнопку мыши (и используйте колесо прокрутки для смены направления, которое вы хотите использовать при телепортировать).
2. Поместите ретикле, где вы хотите телепортироваться.
   * Использование [контроллеров движения](../design/motion-controllers.md): наклон контроллера (на котором удерживается аналоговый стик) для перемещения ретикле.
   * Использование контроллера Xbox. чтобы переместить ретикле, используйте команду " [Взгляните](../design/gaze-and-commit.md) ".
   * С помощью мыши: наведите указатель мыши, чтобы переместить ретикле.
3. Отпустите кнопку, чтобы телепортируйтесь, где была размещена ретикле.

**На практический анализ:**
* Использование [контроллеров движения](../design/motion-controllers.md). Щелкните на аналоговый стик и удерживайте его, а затем переместите аналоговый стик в направлении, которое нужно проанализировать.
* Использование контроллера Xbox: щелкните вниз на левом аналоговый стике и удерживайте его, а затем переместите аналоговый стик в том направлении, в котором вы хотите «пройти».

## <a name="immersive-headset-input-support"></a>Поддержка ввода иммерсивного гарнитуры

[Впечатляющие гарнитуры Windows Mixed Reality](immersive-headset-hardware-details.md) поддерживают несколько типов входных данных для перехода на домашнюю страницу Windows Mixed Reality. HoloLens не поддерживает входные данные для переходов, так как вы физически обходите и видите свою среду. Однако HoloLens [поддерживает входные данные](hardware-accessories.md) для взаимодействия с приложениями.

### <a name="motion-controllers"></a>Контроллеры движений

Лучший опыт Windows Mixed Reality будет работать с [контроллерами движения](../design/motion-controllers.md) Windows Mixed Reality, которые поддерживают шесть средств отслеживания степеней свободы, используя только датчики на гарнитуре — внешние камеры или маркеры не требуются.

Команды навигации ожидается в ближайшее время.

### <a name="gamepad"></a>Геймпад
* **Левый аналоговый стик:**
  * Нажмите и удерживайте левый аналоговый стик вперед, чтобы открыть ретиклеи для [телетранспорта](navigating-the-windows-mixed-reality-home.md#getting-around-your-home) .
  * Коснитесь самого стика влево, вправо или назад, чтобы переместить влево, вправо или назад с небольшим шагом.
  * Щелкните вниз на левом аналоговый стике и удерживайте, а затем переместите аналоговый стик в направлении, которое вы хотите [проанализировать.](navigating-the-windows-mixed-reality-home.md#getting-around-your-home)
* Коснитесь **правого стика** влево или вправо, чтобы повернуть направление на 45 градусов.
* **Нажатие кнопки позволяет** выбрать и действует как жест [касания Air](../design/gaze-and-commit.md#composite-gestures) .
* Нажатие кнопки « **Guide** » открывает [меню «Пуск](navigating-the-windows-mixed-reality-home.md#start-menu) » и действует аналогично жесту [раскрытия](../design/system-gesture.md#bloom) .
* Нажатие **левого и правого триггеров** позволяет увеличить или уменьшить объем приложения, с которым вы работаете в домашней системе.

### <a name="keyboard-and-mouse"></a>Клавиатура и мышь

**Примечание.** Используйте **клавиши Windows + Y** , чтобы переключить мышь между управлением рабочим столом компьютера и домашней средой Windows Mixed Reality.

На домашней странице Windows Mixed Reality:
* Нажатие щелчка **левой** кнопкой мыши выбирает и действует как жест [касания](../design/gaze-and-commit.md#composite-gestures) .
* Нажимая кнопку мыши, вы наводите на нее **право** [на ретикле.](navigating-the-windows-mixed-reality-home.md#getting-around-your-home)
* Нажатие клавиши **Windows** на клавиатуре открывает [меню Пуск](navigating-the-windows-mixed-reality-home.md#start-menu) и действует аналогично жесту [раскрытия](../design/system-gesture.md#bloom) .
* Если [облаками](../design/gaze-and-commit.md) в приложении для двухмерной настольной системы **, щелкните** **правой кнопкой мыши** , чтобы открыть контекстные меню, и прокрутите экран с помощью **колесика прокрутки** (точно так же, как на рабочем столе компьютера).

## <a name="cortana"></a>Кортана

[Кортана](../design/voice-input.md#hey-cortana) — это персональный помощник в Windows Mixed Reality, как и для ПК и телефона. HoloLens имеет встроенный микрофон, но для впечатляющих головных телефонов может потребоваться дополнительный аппаратный уровень. Использование Кортаны для открытия приложений, перезагрузки устройства, поиска в Интернете и других возможностей. Разработчики также могут [интегрировать Кортану](https://dev.windows.com/cortana) в свои впечатления.

Вы также можете использовать голосовые команды, чтобы обойтись дома. Например, наведите указатель на кнопку (с помощью [взгляда](../design/gaze-and-commit.md) или контроллера, в зависимости от устройства) и скажите «SELECT» (выбрать). К другим голосовым командам относятся «Go Home», «больше», «меньше», «близко» и «Face».

## <a name="store-settings-and-system-apps"></a>Хранение, параметры и системные приложения

Windows Mixed Reality имеет несколько встроенных приложений, например:
* **Microsoft Store** для получения приложений и игр
* **Центр обратной связи** для отправки отзывов о системных и системных приложениях
* **Параметры** для настройки системных параметров ([включая обновления сети](../connecting-to-wi-fi-on-hololens.md) и системы)
* **Microsoft ребро** для просмотра веб-сайтов
* **Фотографии** для просмотра фотографий и видео и предоставления к ним доступа
* **Калибровка** (только HoloLens) для настройки интерфейса hololens для текущего пользователя
* **Знакомство с жестами** (HoloLens) или **изучение смешанной реальности** (впечатляющих головных телефонов) для получения сведений об использовании устройства
* **3D Viewer** для украшения мира с помощью содержимого смешанной реальности
* **Портал смешанной реальности** (Настольный компьютер) для настройки и управления головной гарнитурой и потоковой передачи интерактивной предварительной версии представления на гарнитуре для просмотра другими пользователями.
* **Фильмы и телевизор** для просмотра видео 360 и последних фильмов и телепередач
* **Кортана** для всех потребностей вашего виртуального помощника
* **Настольные** (иммерсивное гарнитуры) для просмотра настольного монитора во время работы в иммерсивное головной телефон
* **Обозреватель файлов** Доступ к файлам и папкам, расположенным на устройстве
