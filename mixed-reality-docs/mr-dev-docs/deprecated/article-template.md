*   Методы использования 
*   Преимущества
*   И дон'т'с

Редакционные соглашения:

*   Используйте общий смысл.
*   Не пытайтесь быть забавным.
*   Напишите, используя проверку орфографии и соглашения en-US (США английского языка).
*   **Используйте регистр предложений для заголовков, заголовков разделов и... Ну и все.** Внутри предложений только слова, которые должны быть прописными, являются правильными существительными (как вы узнали в учебном заведении). Названия торговых марок (например, Microsoft HoloLens) являются правильными существительными. Имена функций (например, пространственное сопоставление) не являются правильными существительными, если они не являются охраняемыми товарными знаками.
*   Не создавайте ссылки на форму, [см. раздел](../contributing.md)"Дополнительные сведения". Вместо этого размещайте ссылки в [контекстном](../contributing.md) окне, не направляя читателя в другое место.
