# <a name="release-notes---october-2017"></a>Заметки о выпуске — октябрь 2017 г.

Добро пожаловать в Windows Mixed Reality! в **[Windows 10 Fall Creators Update](https://blogs.windows.com/windowsexperience/2017/10/17/whats-new-windows-10-fall-creators-update/)** выпуске появилась поддержка новых [Windows Mixed Reality впечатляющих головных телефонов](/windows/mixed-reality/discover/immersive-headset-hardware-details) и [контроллеров движения](/windows/mixed-reality/design/motion-controllers). теперь вы можете исследовать новые мировые среды, играть в игры VR и воспроизводить развлечения при подключении к [пк с поддержкой Windows Mixed Reality](./windows-mixed-reality-minimum-pc-hardware-compatibility-guidelines.md).

Windows Mixed Reality гарнитуры и контроллеры движения — это кульминациейность массовых усилий над платформой, а также основной шаг вперед для [платформы Windows Mixed Reality](/windows/mixed-reality/discover/mixed-reality), включая [Microsoft HoloLens](/windows/mixed-reality/hololens-hardware-details). хотя HoloLens не получает обновления для Windows 10 Fall Creators Update, работа над HoloLens не была остановлена. у нас будет множество изучений и ценных сведений, которые будут применяться к нашей недавней работе по Windows Mixed Reality в целом. на самом деле, Windows Mixed Reality впечатляющие головные гарнитуры и контроллеры движения представляют собой удобную точку входа для разработки HoloLens, так как те же интерфейсы api, средства и понятия применяются и к обоим.

чтобы обновить последний выпуск для каждого устройства, откройте приложение **Параметры** , перейдите в раздел **обновление & безопасность**, а затем нажмите кнопку **проверить наличие обновлений** . на Windows 10 компьютере можно также вручную установить Windows 10 Fall Creators Update с помощью [средства Windows создания носителя](https://www.microsoft.com/software-download/windows10).

 **последний выпуск рабочего стола:** Windows 10 desktop октября 2017 (**10.0.16299.15**, Windows 10 Fall Creators Update)<br>
 **последний выпуск для HoloLens:** [Windows 10 Holographic августа 2016](release-notes-august-2016.md) (**10.0.14393.0**, юбилейное обновление Windows 10)

## <a name="introducing-windows-mixed-reality"></a>Введение в Windows Mixed Reality

в Windows 10 Fall Creators Update официально появилась поддержка Windows Mixed Reality гарнитур и контроллеров движения, а также Windows 10 первой пространственной операционной системы мира. Вот некоторые основные моменты:
* **[разнообразные гарнитуры](https://blogs.windows.com/windowsexperience/2017/10/03/how-to-pre-order-your-windows-mixed-reality-headset/)** — Windows Mixed Reality позволяют партнерам предлагать различные типы гарнитур, начиная с $399 долларов сша в комплекте с контроллерами движения.
* **[контроллеры motion](/windows/mixed-reality/design/motion-controllers)** — Windows Mixed Realityные контроллеры движения беспроводные связи с компьютером с помощью Bluetooth и функции, отслеживая шесть степеней свободы, множество методов ввода и имус.
* **[Простота настройки и переносимость](./recommended-adapters-for-windows-mixed-reality-capable-pcs.md)** — настройте и начните работу в течение 10 минут. Впечатляющие головные телефоны используют внутреннее отслеживание для отслеживания движения и контроллеров движения с шестью фридомс. Внешние камеры или маркеры лигхсаусе не требуются.
* **[поддержка более широкого спектра пк](./windows-mixed-reality-minimum-pc-hardware-compatibility-guidelines.md)** — Windows Mixed Reality позволит большему числу пользователей работать с настольными компьютерами, чем когда-либо ранее, благодаря поддержке выбора интегрированных графических плат и пк, начиная с $499 долл. сша.
* **[Windows Mixed Reality home](/windows/mixed-reality/discover/navigating-the-windows-mixed-reality-home)** — первая пространственно-операционная система в мире предоставляет привычную домашнюю среду для выполнения нескольких задач с помощью 2d-приложений, запуска игр и приложений VR и размещения декоративных голограмм.
* **[невероятные игры и приложения в Microsoft Store](https://www.microsoft.com/store/collections/MR-All-ImmersiveContent/)** — от захватывающих развлечений, таких как хулу vr и 360, в компьютерные игры, такие как SUPERHOT VR и солнца, Microsoft Store имеют широкий спектр содержимого для Windows Mixed Reality.
* **[стеамвр ранний доступ](./using-steamvr-with-windows-mixed-reality.md)** . Windows 10 Fall Creators Update обеспечивает поддержку заголовков стеамвр, которые будут воспроизводиться с помощью Windows Mixed Reality гарнитур и контроллеров, что делает самый крупный каталог заголовков VR доступными для Windows Mixed Reality пользователей.

## <a name="known-issues"></a>Известные проблемы

мы работаем над тем, чтобы предоставить отличный опыт работы с Windows Mixed Reality, но мы по-прежнему отслеживаем некоторые известные проблемы. Если вы нашли других пользователей, [отправьте нам отзыв](/windows/mixed-reality/give-us-feedback).

### <a name="desktop-app-in-the-windows-mixed-reality-home"></a>классическое приложение на домашней странице Windows Mixed Reality
* Средство выделения фрагментов не работает в классическом приложении.
* Классическое приложение не сохраняет настройки при повторном запуске.
* если вы используете предварительную версию портала смешанной реальности на настольном компьютере, при открытии классического приложения на Windows Mixed Reality домашней странице вы можете заметить неограниченный результат зеркального отображения. 
* запуск классического приложения может вызвать проблемы с производительностью на компьютерах, не использующих Ultra Windows Mixed Reality; Это не рекомендуется.  
* Классическое приложение может запустить автоматический запуск, так как невидимое окно на рабочем столе имеет фокус. 
* Запрос контроля учетных записей настольных компьютеров сделает гарнитуру черным, пока не завершится запрос.

### <a name="windows-mixed-reality-setup"></a>установка Windows Mixed Reality
* при настройке Windows с подключенным головным телефоном монитор компьютера может быть пустым. отсоедините гарнитуру, чтобы включить вывод на монитор пк для завершения установки Windows.
* При создании границы трассировка может завершиться ошибкой. Если это так, повторите попытку, так как система получит больше информации о вашем пространстве в течение времени.
* если включить или отключить Кортана во время установки Windows Mixed Reality, это изменение будет применено к параметрам настольной Кортана.
* если у вас нет наушников, вы можете пропускать советы при первом посещении Windows Mixed Reality home.
* Bluetooth наушники могут вызвать помехи с помощью контроллеров движения. мы рекомендуем отключать или включать Bluetooth контроллеры во время Windows Mixed Reality сеансов.

### <a name="games-and-apps-from-windows-store"></a>игры и приложения из магазина Windows
* Некоторые высокоресурсоемкие игры, например Форза Моторспортс 6, могут вызвать проблемы с производительностью на менее производительных компьютерах при воспроизведении в Windows Mixed Reality.

### <a name="audio"></a>звук;
* как отмечалось выше, Bluetooth звуковые периферийные устройства плохо работают с Windows Mixed Reality голосовых и пространственных звуковых возможностей. Они также могут негативно сказаться на работе контроллера движения. не рекомендуется использовать Bluetooth гарнитуры с Windows Mixed Reality.
* Вы не можете использовать звуковое устройство, подключенное к (или его части) к гарнитуре для воспроизведения звука, если устройство не надеты. Если у вас есть только одна гарнитура, вам может потребоваться подключить головной телефон к главному компьютеру, а не гарнитуре. если да, то необходимо отключить параметр "переключиться на аудио гарнитуры" в **Параметры**  >    >  **звука и речи** в смешанной реальности.
* Некоторые приложения, в том числе многие из них, запущенные через Стеамвр, могут потерять звук или зависнуть при изменении звукового устройства при запуске или прекращении работы портала смешанной реальности. После открытия приложения портала Mixed Reality перезапустите приложение, чтобы исправить это.
* если перед использованием гарнитуры Windows Mixed Reality на главном компьютере Кортана включен, вы можете потерять симуляцию пространственного звука, применяемую к приложениям, которые вы размещаете вокруг Windows Mixed Reality домашней страницы. обходной обход заключается в том, чтобы включить "Windows Sonic для наушников" для всех звуковых устройств, подключенных к компьютеру, даже для звукового устройства, подключенного к гарнитуре:
   1. Щелкните значок динамика на панели задач рабочего стола и выберите из списка звуковые устройства.
   2. щелкните правой кнопкой мыши значок динамика на панели задач рабочего стола и выберите пункт "Windows Sonic для наушников" в меню "настройка динамика".
   3. Повторите эти действия для всех звуковых устройств (конечных точек).
>[!NOTE]
> - так как наушники и динамики, подключенные к гарнитуре, не будут отображаться, если вы не людьми ее, вам нужно сделать это в окне "классическое приложение" на Windows Mixed Reality домашней странице, чтобы применить этот параметр к звуковому устройству, подключенному к гарнитуре (или интегрированному в гарнитуру).
> - кроме того, можно отключить параметр "разрешить Кортана ответить на" эй Кортана "в **Параметры**  >  **Кортана** на рабочем столе перед запуском Windows Mixed Reality.

* если другое usb-устройство мультимедиа (например, веб-камера) использует один и тот же концентратор usb (внешний или внутренний компьютер) с головной гарнитурой Windows Mixed Reality, в редких случаях звуковая розетка или наушники гарнитуры могут либо иметь звуковой сигнал, либо вообще не иметь звука. Вы можете исправить это в гарнитуре на USB-порт, который не поддерживает тот же центр, что и другое устройство, или отключить или отключить другое USB-устройство мультимедиа.
* в редких случаях USB-концентратор главного компьютера не может обеспечить достаточный уровень электропитания на Windows Mixed Reality гарнитуре, и вы можете заметить, что на наушниках, подключенных к гарнитуре, есть шум.

### <a name="speech"></a>Речь
* Кортана может не воспроизводить звуковые подсказки для прослушивания, обдумывания и передачи звука в команды.
* Кортана в китае, а на рынках японии не отображается текст под Кортана окружности при использовании.
* Кортана может быть задолгой в первый раз, когда она вызывается в сеансе портала смешанной реальности. это можно обойти, убедившись в том, что в разделе **Параметры**  >  **Кортана**  >  **говорить с Кортана** включен параметр "разрешить Кортана ответить на" эй, Кортана ".
* Кортана может выполняться медленнее на компьютерах, на которых не Windows Mixed Reality Ultra пк.
* если системная клавиатура настроена на язык, отличный от языка пользовательского интерфейса в Windows Mixed Reality, использование диктовки с клавиатуры в Windows Mixed Reality приведет к появлению диалогового окна с сообщением об ошибке о том, что диктовка не будет работать из-за отсутствия подключения к Wi-Fi. чтобы устранить эту проблему, убедитесь, что язык системной клавиатуры соответствует языку пользовательского интерфейса Windows Mixed Reality.
* Испания не распознается в качестве рынка, где включено распознавание речи для Windows Mixed Reality.

### <a name="holograms"></a>Holograms (Голограммы)
* если вы поместили большое количество голограмм в Windows Mixed Reality home, некоторые из них могут исчезнуть и появиться снова. чтобы избежать этого, удалите некоторые голограммы из этой области Windows Mixed Reality домашней.

### <a name="motion-controllers"></a>Контроллеры движений
* Иногда при выборе веб-страницы в границе содержимое будет уменьшаться, а не нажато.
* Иногда при выборе ссылки в границе выбор не будет работать.
