# <a name="release-notes---april-2018"></a>Заметки о выпуске — 2018 апреля

**[обновление Windows 10 апрель 2018](https://blogs.windows.com/windowsexperience/2018/04/30/whats-new-in-the-windows-10-april-2018-update)** (также известное как RS4) включает в себя новые функции для HoloLens и Windows Mixed Reality впечатляющих головных телефонов, подключенных к пк. 

чтобы выполнить обновление до последнего выпуска для любого типа устройства, откройте приложение **Параметры** , перейдите в раздел **обновление & безопасность** и нажмите кнопку **проверить наличие обновлений** . на Windows 10 компьютере можно также вручную установить обновление Windows 10 апрель 2018 с помощью [средства создания Windows носителя](https://www.microsoft.com/software-download/windows10).

**последний выпуск для Desktop:** Windows 10 обновлением за апрель 2018 (**10.0.17134.1**)<br>
**последний выпуск для HoloLens:** Windows 10 обновления за апрель 2018 (**10.0.17134.80**)<br>

## <a name="new-features-for-windows-mixed-reality-immersive-headsets"></a>новые функции для Windows Mixed Reality иммерсивное гарнитуры

Windows 10 обновление за апрель 2018 включает множество улучшений использования Windows Mixed Reality иммерсивное (VR) гарнитуры на настольном пк, например: 

* **новые среды для домашней страницы смешанной реальности** — выбор между клифф дома и новой средой скилофт путем выбора **мест** на меню. Мы также добавили [экспериментальную функцию](/windows/mixed-reality/design/add-custom-home-environments) , которая позволит использовать созданные вами пользовательские среды.
* **Быстрый доступ к захвату смешанной реальности** . Возьмите фотографии смешанной реальности с помощью контроллера движения в средах и приложениях, но не записывает содержимое, защищенное DRM. нажмите кнопку Windows, а затем выберите триггер. .
* **новые параметры для запуска и изменения размера содержимого** — приложения теперь автоматически помещаются перед вами при запуске из меню. Теперь можно изменить размер 2D-приложения, перетащив края и углы окна.
* вы можете **легко перейти к содержимому с помощью команды "телепортируйтесь"** , быстро телепортироваться перед содержимым в Windows Mixed Reality home, облаками в содержимом и указав "телепортируйтесь."
* **[Анимированные средства запуска 3D-приложений](/windows/mixed-reality/distribute/creating-3d-models-for-use-in-the-windows-mixed-reality-home#animation-guidelines) и [декоративные трехмерные объекты](/windows/mixed-reality/distribute/enable-placement-of-3d-models-in-the-home) для дома в смешанной реальности**. теперь можно добавить анимацию в средства запуска 3d-приложений и разрешить пользователям размещать декоративные трехмерные модели из веб-страницы или 2d-приложения в Windows Mixed Reality домашней странице.
* **[улучшения Windows Mixed Reality для стеамвр](/windows/mixed-reality/develop/porting-apps/updating-your-steamvr-application-for-windows-mixed-reality)** -Windows Mixed Reality для стеамвр — это "ранний доступ" с новыми обновлениями, включая: хаптик отзывы при использовании контроллеров движения, улучшенную производительность и надежность, а также улучшения внешнего вида контроллеров движения в стеамвр.
* **Другие улучшения** . обновленные автоматические параметры производительности обеспечивают более оптимизированный интерфейс (можно [вручную переопределить](#visual-quality) этот параметр). Теперь программа установки предоставляет более подробные сведения о распространенных проблемах совместимости с контроллерами USB 3,0 и графическими картами.

## <a name="known-issues"></a>Известные проблемы

мы работаем над тем, чтобы предоставить отличный опыт работы с Windows Mixed Reality, но мы по-прежнему отслеживаем некоторые известные проблемы. Если вы нашли других пользователей, отправьте [нам отзыв](/windows/mixed-reality/give-us-feedback).

### <a name="hololens"></a>HoloLens

#### <a name="after-update"></a>После обновления

После обновления с RS1 на RS4 в HoloLens можно заметить следующие проблемы:
* **пин-коды сброшены** . приложения 3x3, закрепленные на меню будут сброшены до значений по умолчанию после обновления. 
* **Приложения и размещенные голограммы сброшены** . приложения, размещенные в вашем мире, будут удалены после обновления и должны быть переустановлены в течение всего пространства. 
* **Центр обратной связи может не запуститься сразу** же после обновления. это займет несколько минут, прежде чем вы сможете запустить некоторые входящие приложения, такие как центр отзывов, в то время как они обновляются. 
* **необходимо повторно синхронизировать корпоративные Wi-Fi сертификаты** . мы изучаем проблему, которая требует, чтобы HoloLens подключались к другой сети, чтобы корпоративные сертификаты были повторно синхронизированы с устройством, прежде чем оно сможет снова подключиться к корпоративным сетям с помощью сертификатов. 
* **H. 265 HEVC воспроизведение видео не работает** — приложения, пытающиеся воспроизвести видео h. 265, получат сообщение об ошибке. чтобы устранить эту проблему, перейдите на [портал устройств Windows](/windows/mixed-reality/develop/platform-capabilities-and-apis/using-the-windows-device-portal), выберите **приложения** на левой навигационной панели и **удалите** приложение HEVC. затем установите последнюю [Расширение для видео HEVC](https://www.microsoft.com/p/hevc-video-extensions/9nmzlz57r3t7) из Microsoft Store. Мы изучаем проблему. 

#### <a name="for-developers-updating-hololens-apps-for-devices-running-windows-10-april-2018-update"></a>для разработчиков: обновление HoloLens приложений для устройств под Windows 10 апреля, обновление 2018

наряду с отличным списком [новых возможностей](#new-features-for-hololens)Windows 10 обновлением за апрель 2018 (RS4) для HoloLens применяют некоторые поведения кода, которые не были доступны в предыдущих версиях:
* **запросы на разрешение на использование конфиденциальных ресурсов (камера, микрофон и т. д.)** — RS4 на HoloLens будут применять запросы разрешений для приложений, которые обращаются к конфиденциальным ресурсам, таким как камера или микрофон. RS1 на HoloLens не задали эти запросы, поэтому, если приложение предполагает немедленный доступ к этим ресурсам, оно может завершиться сбоем в RS4 (даже если пользователь предоставит разрешение на запрошенный ресурс). Дополнительные сведения см. в соответствующей [статье объявления возможностей приложений UWP](/windows/uwp/packaging/app-capability-declarations) .
* **вызовы приложений за пределами собственного** RS4 в HoloLens будут обеспечивать правильное использование [ *Windows.System. Класс запуска*](/uwp/api/Windows.System.Launcher) для запуска другого приложения самостоятельно. Например, мы видели проблемы с приложениями, вызывающими *Windows.SysTEM. Launcher. Лаунчурифорресултсасинк* из потока, отличного от ASTA (UI). это будет выполнено в RS1 на HoloLens, но RS4 требует выполнения вызова в потоке пользовательского интерфейса.

### <a name="windows-mixed-reality-on-desktop"></a>Windows Mixed Reality на рабочем столе

#### <a name="visual-quality"></a>Визуальное качество

* если вы заметили, что после обновления Windows 10 от апреля 2018, что графика более размыта, чем раньше, или что поле представления на гарнитуре выглядит меньше, то параметр автоматического повышения производительности может быть изменен для поддержания достаточной частоты кадров на менее мощной графической карте (например, Nvidia 1050). вы можете вручную переопределить этот вариант (если выбрать), перейдя в **Параметры > смешанная реальность > отображение гарнитуры > возможности > изменить** и изменить значение "автоматически" на "90 гц". можно также попытаться изменить **качество визуального** элемента (на той же странице Параметры) на «высокая».

#### <a name="windows-mixed-reality-setup"></a>установка Windows Mixed Reality

* при настройке Windows с подключенным головным телефоном монитор компьютера может быть пустым. отсоедините гарнитуру, чтобы включить вывод на монитор пк для завершения установки Windows.
* если у вас нет наушников, вы можете пропускать советы при первом посещении Windows Mixed Reality home.
* другие Bluetooth устройства могут вызвать помехи при использовании контроллеров движения. если в контроллерах motion возникают проблемы с подключением, связыванием и отслеживанием, убедитесь, что Bluetooth радио (если внешний ключ) подключен к неограниченному расположению, а не сразу рядом с другим Bluetoothным ключом. также попробуйте включить другие Bluetooth периферийные устройства во время Windows Mixed Reality сеансов, чтобы узнать, помогает ли он.

#### <a name="games-and-apps-from-the-microsoft-store"></a>Игры и приложения из Microsoft Store

* Некоторые ресурсоемкие игры, такие как Форза МОТОРСПОРТ 7, могут вызвать проблемы с производительностью на менее производительных компьютерах при воспроизведении в Windows Mixed Reality.

#### <a name="audio"></a>звук;

* если на главном компьютере Кортана включен, прежде чем использовать гарнитуру Windows Mixed Reality, вы можете потерять имитацию звука, примененную к приложениям, размещенным вокруг Windows Mixed Reality домашней страницы. 
   * обходной обход заключается в том, чтобы включить "Windows Sonic для наушников" для всех звуковых устройств, подключенных к компьютеру, даже для звукового устройства, подключенного к гарнитуре:
      1. Щелкните значок динамика на панели задач рабочего стола и выберите из списка звуковые устройства.
      2. щелкните правой кнопкой мыши значок динамика на панели задач рабочего стола и выберите пункт "Windows Sonic для наушников" в меню "настройка динамика".
      3. Повторите эти действия для всех звуковых устройств (конечных точек).
   * кроме того, можно отключить параметр "разрешить Кортана ответить на" эй, Кортана "в **Параметры**  >  **Кортана** на рабочем столе перед запуском Windows Mixed Reality.
* если другое usb-устройство мультимедиа (например, веб-камера) использует один и тот же концентратор usb (внешний или внутренний компьютер) с головной гарнитурой Windows Mixed Reality, в редких случаях звуковая розетка или наушники гарнитуры могут либо иметь звуковой сигнал, либо вообще не иметь звука. Вы можете исправить это в гарнитуре на USB-порт, который не поддерживает тот же центр, что и другое устройство, или отключить или отключить другое USB-устройство мультимедиа.
* в редких случаях USB-концентратор главного компьютера не может обеспечить достаточный уровень электропитания на Windows Mixed Reality гарнитуре, и вы можете заметить, что на наушниках, подключенных к гарнитуре, есть шум.

#### <a name="holograms"></a>Holograms (Голограммы)

* если вы поместили большое количество голограмм в Windows Mixed Reality home, некоторые из них могут исчезнуть и появиться снова. чтобы избежать этого, удалите некоторые голограммы из этой области Windows Mixed Reality домашней.

#### <a name="motion-controllers"></a>Контроллеры движений

* Если входные данные не направляются на гарнитуру, контроллер движения будет кратковременно исчезать при удержании рядом с границей комнаты. Нажмите клавиши Win + Y, чтобы убедиться, что на настольном мониторе есть синий баннер, позволяющий решить эту проблему. 
* иногда при щелчке на веб-странице в Microsoft Edge содержимое будет уменьшено, а не нажато.

#### <a name="desktop-app-in-the-windows-mixed-reality-home"></a>классическое приложение на домашней странице Windows Mixed Reality

* Средство выделения фрагментов не работает в классическом приложении.
* Классическое приложение не сохраняет настройки при повторном запуске.
* если вы используете предварительную версию портала смешанной реальности на настольном компьютере, при открытии классического приложения на Windows Mixed Reality домашней странице вы можете заметить неограниченный результат зеркального отображения. 
* запуск классического приложения может вызвать проблемы с производительностью на компьютерах, не использующих Ultra Windows Mixed Reality; Это не рекомендуется.  
* Классическое приложение может запустить автоматический запуск, так как невидимое окно на рабочем столе имеет фокус. 
* Запрос контроля учетных записей настольных компьютеров сделает гарнитуру черным, пока не завершится запрос.

#### <a name="windows-mixed-reality-for-steamvr"></a>Windows Mixed Reality для стеамвр

* после обновления может потребоваться запустить портал Mixed Reality, чтобы убедиться, что необходимые обновления программного обеспечения для Windows 10 обновления от апреля 2018 до запуска стеамвр. 
* для стеамвр можно использовать последнюю версию Windows Mixed Reality для поддержания совместимости с обновлением Windows 10 апреля 2018. убедитесь, что служба автоматического обновления включена для Windows Mixed Reality для стеамвр, которая находится в разделе "программное обеспечение" библиотеки в Steam.  

#### <a name="other-issues"></a>Другие проблемы

>[!IMPORTANT]
>в ранней версии Windows 10 обновления за апрель 2018, отправленной для участников программы предварительной оценки (версия 17134,5), отсутствовала часть программного обеспечения, необходимое для запуска Windows Mixed Reality. Рекомендуется исключить эту версию, если используется Windows Mixed Reality. 

мы определили снижение производительности при использовании Surface Book 2 в первоначальном выпуске этого обновления (10.0.17134.1), который мы работаем над исправлением в предстоящем обновлении. Мы рекомендуем подождать, пока она не будет обновлена вручную, или подождать, пока не будет выполнено развертывание обновления.
