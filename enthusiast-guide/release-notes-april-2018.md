# <a name="release-notes---april-2018"></a>Заметки о выпуске — 2018 апреля

**[Обновление Windows 10 от апреля 2018](https://blogs.windows.com/windowsexperience/2018/04/30/whats-new-in-the-windows-10-april-2018-update)** (также известное как RS4) включает новые функции для захватывающих головных телефонов HoloLens и Windows Mixed Reality, подключенных к ПК. 

Чтобы выполнить обновление до последнего выпуска для любого типа устройства, откройте приложение " **Параметры** ", перейдите в раздел " **Обновление & безопасность** ", а затем нажмите кнопку " **проверить обновления** ". На компьютере с Windows 10 можно также вручную установить обновление Windows 10 от апреля 2018 с помощью [средства создания Windows Media](https://www.microsoft.com/software-download/windows10).

**Последний выпуск для Desktop:** Обновление Windows 10 от апреля 2018 ( **10.0.17134.1** )<br>
**Последний выпуск для HoloLens:** Обновление Windows 10 от апреля 2018 ( **10.0.17134.80** )<br>

## <a name="new-features-for-windows-mixed-reality-immersive-headsets"></a>Новые возможности для впечатляющих головных телефонов Windows Mixed Reality

Обновление Windows 10 от апреля 2018 г. включает множество улучшений использования головных телефонов Windows Mixed Reality (VR) с настольным ПК, например: 

* **Новые среды для дома "смешанная реальность** ". Теперь вы можете выбрать между Клифф дома и новой средой скилофт, выбрав пункт " **места** в меню" Пуск ". Мы также добавили [экспериментальную функцию](https://docs.microsoft.com/windows/mixed-reality/design/add-custom-home-environments) , которая позволит использовать созданные вами пользовательские среды.
* **Быстрый доступ к записи смешанной реальности** . Теперь вы можете брать фотографии смешанной реальности с помощью контроллера движения. Нажмите кнопку Windows, а затем выберите триггер. Это работает в разных средах и приложениях, но не будет записывать содержимое, защищенное DRM.
* **Новые параметры для запуска и изменения размера содержимого** — приложения теперь автоматически помещаются перед вами при запуске из меню "Пуск". Теперь можно изменить размер 2D-приложения, перетащив края и углы окна.
* **Простое переход к содержимому с помощью команды "телепортируйтесь"** . Теперь вы можете быстро телепортироваться к содержимому в домашней среде Windows Mixed Reality, облаками на содержимое и указав "телепортируйтесь".
* **[Анимированные запуски 3D-приложений](https://docs.microsoft.com/windows/mixed-reality/distribute/creating-3d-models-for-use-in-the-windows-mixed-reality-home#animation-guidelines) и [декоративные трехмерные объекты](https://docs.microsoft.com/windows/mixed-reality/distribute/enable-placement-of-3d-models-in-the-home) для домашней страницы смешанной реальности** . Теперь вы можете добавить анимацию в средства запуска 3D-приложений и позволить пользователям размещать декоративные трехмерные модели с веб-страниц или 2D-приложений в домашней среде Windows Mixed Reality.
* **[Улучшения в Windows Mixed Reality для стеамвр](https://docs.microsoft.com/windows/mixed-reality/develop/porting-apps/updating-your-steamvr-application-for-windows-mixed-reality)** — в Windows Mixed Reality для стеамвр есть новые обновления, включая отзывы о хаптик при использовании контроллеров движения, повышении производительности и надежности, а также улучшения внешнего вида контроллеров движения в стеамвр.
* **Другие улучшения** — автоматические параметры производительности были обновлены для обеспечения более оптимизированного интерфейса (можно [вручную переопределить](#visual-quality) этот параметр). Теперь программа установки предоставляет более подробные сведения о распространенных проблемах совместимости с контроллерами USB 3,0 и графическими картами.

## <a name="known-issues"></a>Известные проблемы

Мы работаем над тем, чтобы предоставить отличный опыт Windows Mixed Reality, но мы по-прежнему отслеживаем некоторые известные проблемы. Если вы нашли других пользователей, отправьте [нам отзыв](https://docs.microsoft.com/windows/mixed-reality/give-us-feedback).

### <a name="hololens"></a>HoloLens

#### <a name="after-update"></a>После обновления

После обновления с RS1 на RS4 в HoloLens можно заметить следующие проблемы:
* **ПИН-коды сброшены** . приложения 3x3, закрепленные в меню "Пуск", будут сброшены до значений по умолчанию после обновления. 
* **Приложения и размещенные голограммы сброшены** . приложения, размещенные в вашем мире, будут удалены после обновления и должны быть переустановлены в течение всего пространства. 
* **Центр обратной связи может не запуститься сразу** же после обновления. это займет несколько минут, прежде чем вы сможете запустить некоторые входящие приложения, такие как центр отзывов, в то время как они обновляются. 
* **Корпоративные сертификаты Wi-Fi необходимо повторно синхронизировать** — мы изучаем проблему, которая требует подключения HoloLens к другой сети, чтобы корпоративные сертификаты были повторно синхронизированы с устройством, прежде чем оно сможет восстановить подключение к корпоративным сетям с помощью сертификатов. 
* **H. 265 HEVC видео не работает** — приложения, пытающиеся воспроизвести видео h. 265, получат сообщение об ошибке. Чтобы устранить эту проблему, перейдите на [портал устройств Windows](https://docs.microsoft.com/windows/mixed-reality/develop/platform-capabilities-and-apis/using-the-windows-device-portal), выберите **приложения** на левой навигационной панели и **удалите** приложение HEVC. Затем установите последнюю версию [расширения видео HEVC](https://www.microsoft.com/p/hevc-video-extensions/9nmzlz57r3t7) из Microsoft Store. Мы изучаем проблему. 

#### <a name="for-developers-updating-hololens-apps-for-devices-running-windows-10-april-2018-update"></a>Для разработчиков: обновление приложений HoloLens для устройств под Windows 10 апреля с обновлением 2018

Наряду с отличным списком [новых функций](#new-features-for-hololens), обновление Windows 10 от апреля 2018 (RS4) для HoloLens обеспечивает выполнение некоторых поведений кода, которые не были выполнены в предыдущих версиях.
* **Запросы разрешений на использование конфиденциальных ресурсов (камера, микрофон и т. д.)** -RS4 в HoloLens будет применять запросы разрешений для приложений, которые будут получать доступ к конфиденциальным ресурсам, таким как камера или микрофон. RS1 в HoloLens не применяет эти запросы, поэтому, если приложение предполагает немедленный доступ к этим ресурсам, оно может аварийно завершить работу в RS4 (даже если пользователь предоставил разрешение на запрошенный ресурс). Дополнительные сведения см. в соответствующей [статье объявления возможностей приложений UWP](https://docs.microsoft.com/windows/uwp/packaging/app-capability-declarations) .
* **Вызовы приложений за пределами вашего собственного** параметра-RS4 в HoloLens будут обеспечивать правильное использование [ *Windows.SysTEM. Класс запуска*](https://docs.microsoft.com/uwp/api/Windows.System.Launcher) для запуска другого приложения самостоятельно. Например, мы видели проблемы с приложениями, вызывающими *Windows.SysTEM. Launcher. Лаунчурифорресултсасинк* из потока, отличного от ASTA (UI). Это будет выполнено в RS1 в HoloLens, но RS4 требует выполнения вызова в потоке пользовательского интерфейса.

### <a name="windows-mixed-reality-on-desktop"></a>Windows Mixed Reality на настольных компьютерах

#### <a name="visual-quality"></a>Визуальное качество

* Если вы видите после обновления Windows 10 от апреля 2018, что графика более размыта, чем раньше, или что поле представления на гарнитуре выглядит меньше, возможно, параметр автоматического повышения производительности был изменен, чтобы поддерживать достаточное количество кадров на менее мощной графической карте (например, NVIDIA 1050). Вы можете вручную переопределить это (если вы выбираете), перейдя к **параметрам > Смешанная реальность > отображение гарнитуры > параметры работы > изменить** и измените значение "автоматически" на "90 Гц". Можно также попытаться изменить **качество визуального** элемента (на той же странице параметров) на «высокий».

#### <a name="windows-mixed-reality-setup"></a>Установка Windows Mixed Reality

* При настройке Windows с подключением гарнитуры монитор компьютера может быть пустым. Отсоедините гарнитуру, чтобы включить вывод на монитор ПК для завершения установки Windows.
* Если у вас нет наушников, вы можете пропускать дополнительные советы при первом посещении домашней страницы Windows Mixed Reality.
* Другие устройства Bluetooth могут вызвать помехи при использовании контроллеров движения. Если в контроллерах Motion возникают проблемы подключения, связывания и отслеживания, убедитесь, что радиомодуль Bluetooth (если внешний аппаратный ключ) подключен к неограниченному расположению, а не сразу рядом с другим аппаратным модулем Bluetooth. Также попробуйте включить другие периферийные устройства Bluetooth во время сеансов Windows Mixed Reality, чтобы узнать, помогает ли он.

#### <a name="games-and-apps-from-the-microsoft-store"></a>Игры и приложения из Microsoft Store

* Некоторые высокоресурсоемкие игры, например Форза МОТОРСПОРТ 7, могут вызвать проблемы с производительностью на менее производительных компьютерах при воспроизведении в Windows Mixed Reality.

#### <a name="audio"></a>звук;

* Если вы включили Кортану на главном компьютере до использования головного телефона Windows Mixed Reality, вы можете потерять пространственный симуляцию, примененную к приложениям, размещаемым на домашней странице Windows Mixed Reality. 
   * Обходной обход заключается в том, чтобы включить "Windows Sonic для наушников" на всех подключенных к компьютеру звуковых устройствах, даже на устройстве, подключенном к гарнитуре:
      1. Щелкните значок динамика на панели задач рабочего стола и выберите из списка звуковые устройства.
      2. Щелкните правой кнопкой мыши значок динамика на панели задач рабочего стола и выберите пункт "Windows Sonic для наушников" в меню "Настройка динамика".
      3. Повторите эти действия для всех звуковых устройств (конечных точек).
   * Другой вариант — Отключить "разрешить Кортане отвечать на Привет, Кортана" в **настройках**  >  **кортаны** на рабочем столе перед запуском Windows Mixed Reality.
* Если другое USB-устройство мультимедиа (например, веб-камера) использует один и тот же концентратор USB (как внешний, так и внутренний) с головной гарнитурой Windows Mixed Reality, в редких случаях звуковая розетка или наушники гарнитуры могут либо иметь звуковой сигнал, либо вообще не иметь звука. Вы можете исправить это по гарнитуре на USB-порт, который не использует тот же центр, что и другое устройство, или отключить или отключить другое USB-устройство мультимедиа.
* В редких случаях USB-концентратор главного компьютера не может обеспечить достаточный уровень электропитания для гарнитуры Windows Mixed Reality, и вы можете заметить, что у наушников, подключенных к гарнитуре, имеется достаточное количество помех.

#### <a name="holograms"></a>Holograms (Голограммы)

* Если вы поместили большое количество голограмм на домашнюю страницу Windows Mixed Reality, некоторые из них могут исчезнуть и отображаться в процессе поиска. Чтобы избежать этого, удалите некоторые голограммы из этой области домашней страницы Windows Mixed Reality.

#### <a name="motion-controllers"></a>Контроллеры движений

* Если входные данные не направляются на гарнитуру, контроллер движения будет кратковременно исчезать при удержании рядом с границей комнаты. Нажмите клавиши Win + Y, чтобы убедиться, что на настольном мониторе есть синий баннер, позволяющий решить эту проблему. 
* Иногда при щелчке веб-страницы в Microsoft ребро содержимое будет уменьшено, а не нажато.

#### <a name="desktop-app-in-the-windows-mixed-reality-home"></a>Классическое приложение на домашней странице Windows Mixed Reality

* Средство "ножницы" не работает в классическом приложении.
* Классическое приложение не сохраняет настройки при повторном запуске.
* Если вы используете предварительную версию портала смешанной реальности на рабочем столе, при открытии классического приложения на домашней странице Windows Mixed Reality вы можете заметить неограниченное воздействие зеркального отображения. 
* Запуск классического приложения может вызвать проблемы с производительностью на компьютерах, не поддерживающих Ultra Windows Mixed Reality. Это не рекомендуется.  
* Классическое приложение может запустить автоматический запуск, так как невидимое окно на рабочем столе имеет фокус. 
* Запрос контроля учетных записей настольных компьютеров сделает гарнитуру черным, пока не завершится запрос.

#### <a name="windows-mixed-reality-for-steamvr"></a>Windows Mixed Reality для Стеамвр

* После обновления может потребоваться запустить портал Mixed Reality, чтобы убедиться, что необходимые обновления для Windows 10 от апреля 2018 были завершены перед запуском Стеамвр. 
* Вы должны быть в последней версии Windows Mixed Reality, чтобы Стеамвр оставалась совместимой с обновлением Windows 10 от апреля 2018. Убедитесь, что служба автоматического обновления включена для Windows Mixed Reality для Стеамвр, которая находится в разделе "программное обеспечение" библиотеки в Steam.  

#### <a name="other-issues"></a>Другие проблемы

>[!IMPORTANT]
>В ранней версии обновления Windows 10 от апреля 2018, отправленной для участников программы предварительной оценки (версия 17134,5), отсутствовало программное обеспечение, необходимое для запуска Windows Mixed Reality. Рекомендуется исключить эту версию, если используется Windows Mixed Reality. 

Мы обнаружили регрессию производительности при использовании контактной книги 2 в первоначальном выпуске этого обновления (10.0.17134.1), который мы работаем над исправлением в предстоящем обновлении. Мы рекомендуем подождать, пока она не будет обновлена вручную, или подождать, пока не будет выполнено развертывание обновления.
