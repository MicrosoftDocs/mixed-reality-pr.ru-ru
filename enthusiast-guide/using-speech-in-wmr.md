# <a name="using-speech-in-windows-mixed-reality"></a>Использование речи в Windows Mixed Reality

Вы можете использовать свой голоса для ускорения работы с Windows Mixed Reality. Благодаря быстрому фото, открытию приложения, даже при переносе без контроллера, все слова не удаляются. Чтобы легко вводить текст, попробуйте использовать режим диктовки на клавиатуре Mixed Reality. 

Возникли проблемы с голосовыми функциями? [Получить помощь](using-wmr-faq.yml#speech-commands-aren-t-working)

<!-- NEED VIDEO: https://support.microsoft.com/en-us/help/4041322/windows-10-speech-in-windows-mixed-reality -->

> [!NOTE]
> * Если речь включена, Windows Mixed Reality всегда прослушивается. При подключении к Интернету мы отправляем все, что вы говорите в облако, чтобы службы Microsoft Speech Services могли распознавать еще больше команд.
> * Речевые команды не поддерживаются на всех языках. Подробнее
> * Гарнитуры и динамики Bluetooth не поддерживаются в Windows Mixed Reality.

## <a name="see-it-say-it"></a>Модель "Посмотри на это, произнеси это"

Если на домашней странице Windows Mixed Reality отображается слово, его часто можно использовать в качестве речевой команды. Например, просто скажите имя кнопки, чтобы выбрать ее. Если имя не отображается, наведите указатель на контроллер движения на кнопке, чтобы узнать, что сказать. Для игровых планшетов Xbox загляните на кнопку.

## <a name="general-speech-commands"></a>Общие голосовые команды

Используйте следующие голосовые команды в Windows Mixed Reality, чтобы быстрее приступить к работе. Некоторые команды используют курсор «взгляд», который вы выберете «SELECT».

| Требуемое действие | Скажем |
| --- | --- |
| Выберите пункт | Скажите «SELECT», чтобы открыть курсор «взгляд». Затем Превратите в голову курсор на то, что нужно выбрать, и скажите "выбрать" еще раз. |
| Открытие меню "Пуск" | Переход в меню "Пуск" |
| Покинуть иммерсивное приложение | Скажите "перейти к началу", чтобы открыть меню быстрых действий, а затем скажите "Домашняя страница" Mixed Reality ". |
| Включить или выключить фонарик | Фонарик on/фонарик Off |
| Телепортируйтесь | Превратите свой заголовок в нужное место, а затем скажите «телепортируйтесь». (Для более точной нацеливания сначала скажите "Select", чтобы открыть курсор Взгляните, а затем скажите "телепортируйтесь.") |
| Повернуть влево или вправо | Повернуть влево или вправо |
| Повернуть на 180 градусов | Повернись |
| Переместить вперед | Перемещение вперед или назад |
| Резервное копирование | Назад или вперед |
| Переместить влево | Переместить влево или влево |
| Переместить вправо | Переместить вправо/разобрать вправо |

Скажите "Привет, Кортана!", а затем используйте одну из следующих команд:

| Требуемое действие | Скажем |
| --- | --- |
| Узнайте, что можно сказать о Кортане | Что я могу сказать? |
| Увеличение или уменьшение объема | Включение и отключение тома |
| Включить или отключить звук | Включить или отключить звук |
| Запуск приложения | Launch [имя приложения] |
| Открытие веб-сайта в Microsoft ребро | Откройте [имя веб-сайта] (например, Open bing.com). |
| Сделать фотографию | Сделать снимок |
| Начать запись видео | Начать запись |
| Отмена записи видео | Остановить запись |
| Показывать время | Сколько времени? |
| Открытие меню "Пуск" | Открыть меню "Пуск" |
| Установка таймера | Установка таймера |
| Установка напоминания | Установка напоминания |

> [!NOTE]
> * Кортана доступна не во всех регионах и языках. [Подробнее](https://support.microsoft.com/help/4026948).
> * Если Кортана не отвечает на команду "Привет, Кортана", выберите **параметры > конфиденциальность > речи** и включите распознавание речи в сети.
> * Если вы выключите Кортану, голосовые команды "Привет, Кортана!" будут недоступны, но вы по-прежнему сможете использовать другие команды (например, "Select" и "телепортируйтесь").

## <a name="keyboard-dictation"></a>Диктовка с клавиатуры

Переключение в режим диктовки в любое время, когда клавиатура активна для простого ввода текста. Чтобы приступить к работе, выберите микрофон на клавиатуре или просто скажите «начать диктовку».

> [!NOTE]
> Клавиатура Mixed Reality доступна только на английском языке, но вы можете использовать диктовку на любом из поддерживаемых [языков Windows Mixed Reality](other-questions.md#what-languages-are-supported-in-windows-mixed-reality).

### <a name="keyboard-dictation-commands"></a>Команды диктовки с клавиатуры

| Требуемое действие | Скажем |
| --- | --- |
| Закрыть клавиатуру | Закрыть |
| Начать диктовку | Начать диктовку |
| Прерывать диктовку | Прерывать диктовку |
| Удаление того, что вы определили | Удалите этот |
| Выберите все в поле диктовка. | Выбрать все |

### <a name="punctuation"></a>Пунктуация

Необходимо указать имена знаков препинания, которые вы хотите использовать. Например, вы можете сказать: « **запятые** , что вы хотите увидеть в качестве вопросительного **знака**».

Ниже приведены ключевые слова пунктуации, которые можно использовать:

* Точка, запятая, вопросительный знак, восклицательный знак или восклицательный знак
* Новая строка/новый абзац
* Точка с запятой, двоеточие
* Открывающие кавычки, закрывающие кавычки
* Хэш-тег, смайлик/смайлик, хмурого, мультик
* Доллар, процент

Иногда бывает полезно выполнить некоторые действия, например адреса электронной почты. Например, для диктовки example@outlook.com вы можете сказать: «E X A M P L E в Outlook dot-com».

Чтобы отключить диктовку, нажмите кнопку **Готово**.
