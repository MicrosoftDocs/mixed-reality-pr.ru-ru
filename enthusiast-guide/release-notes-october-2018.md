# <a name="release-notes---october-2018"></a>Заметки о выпуске — октябрь 2018 г.

**[обновление Windows 10 за октябрь 2018 г.](https://blogs.windows.com/windowsexperience/2018/10/02/find-out-whats-new-in-windows-and-office-in-october/)** (также известный как RS5) включает новые функции для HoloLens и Windows Mixed Realityе впечатляющих головных телефонов, подключенных к пк. 

чтобы выполнить обновление до последнего выпуска на HoloLens или пк (для Windows Mixed Realityных (VR) головных телефонов, откройте приложение **Параметры** , перейдите в раздел **обновление & безопасность**, а затем нажмите кнопку **проверить наличие обновлений** . на Windows 10 компьютере можно также вручную установить обновление Windows 10 за октябрь 2018 г. с помощью [средства Windows создания носителя](https://www.microsoft.com/software-download/windows10).

**последний выпуск рабочего стола:** обновление Windows 10 за октябрь 2018 г. (**10.0.17763.107**)<br>
**последний выпуск для HoloLens:** обновление Windows 10 за октябрь 2018 г. (**10.0.17763.134**)<br>

## <a name="new-features-for-windows-mixed-reality-immersive-headsets"></a>новые функции для Windows Mixed Reality иммерсивное гарнитуры

обновление Windows 10 за октябрь 2018 г. включает множество улучшений использования Windows Mixed Reality иммерсивного (VR) гарнитур с настольным пк.

### <a name="for-everyone"></a>Для всех

* **Смешанная реальность фонарик** — откройте портал в реальном мире, чтобы найти свою клавиатуру, просмотреть сведения о своей обстановке, или взгляните на окружающую подстановку, не удаляя гарнитуру! вы можете включить смешанную реальность фонарик из меню, нажав Windows + захватить на контроллере движения или выполнив команду "фонарик on/off". Наведите указатель на контроллер в том направлении, что вы хотите видеть, например с помощью фонарик в темной.

    ![Смешанная реальность фонарик](images/mr-flashlight.png)

* **Новые приложения и способы запуска содержимого на домашней странице Mixed Reality**
    * если вы используете [Windows Mixed Reality для стеамвр](./using-steamvr-with-windows-mixed-reality.md), заголовки стеамвр теперь отображаются в меню и средства запуска приложений для каждого из них можно разместить на домашней странице Mixed Reality.
    
        ![Средства запуска приложений Стеамвр](images/steamvr-launchers.png)
        
    * Новое приложение *360 видео* для обнаружения регулярно проверенных видеороликов 360.
    * Новое *демонстрационное приложение вебвр* для обнаружения регулярно проверенного набора вебврных возможностей.
    * при первом Windows Mixed Reality клиенты будут входить в клифф House и найти его предварительно заполненные средства запуска 3d-приложений для некоторых из наших любимых приложений и игр из Microsoft Store.
    * в Microsoft Edge windows теперь есть кнопка *общего доступа* .
* **меню быстрые действия** — изнутри приложения смешанной среды, можно нажать кнопку Windows, чтобы открыть новое меню быстрых действий с простым доступом к *меню стеамвр*, *видеозаписи фотографий и видео*, *фонарик* и *дома*.
* **поддержка ранец пк** Windows Mixed Reality — головные гарнитуры (VR) работают на ранец компьютерах без необходимости использования эмулятора экрана после завершения установки.
* **новые функции звука** . теперь вы можете отразить звук от Windows Mixed Realityного интерфейса на наушнике (или наушниках) на гарнитуре *и* звуковом устройстве, подключенном к компьютеру (например, к внешним динамикам). Мы также добавили визуальный индикатор для уровня громкости на дисплее вашей гарнитуры.
* **Другие улучшения**
    * обновления портала смешанной реальности теперь доставляются через Microsoft Store, что позволяет быстрее обновляться между основными Windowsными выпусками. обратите внимание, что это относится только к классу desktop, а Windows Mixed Reality интерфейс гарнитуры по-прежнему обновляется с ос. 
    * когда гарнитуры переходят в спящий режим, Windows Mixed Reality приложения приостанавливаются, а не прерываются (пока не будет закрыт портал смешанной реальности).
    
### <a name="for-developers"></a>Для разработчиков

* **[отслеживание qr-кода](/windows/mixed-reality/develop/platform-capabilities-and-apis/qr-code-tracking)** . включите отслеживание qrного кода в приложении смешанной реальности, что позволяет Windows Mixed Reality иммерсивное (VR) гарнитуры проверять наличие QR-кодов и передавать их в интересующие приложения.
* **Поддержка аппаратных устройств DRM для впечатляющих приложений** . разработчики теперь могут запрашивать аппаратно защищенные текстуры буфера, если они поддерживаются аппаратным обеспечением экрана, что позволяет приложениям использовать аппаратно-защищенное содержимое из таких источников, как PlayReady.
* **[интеграция пользовательского интерфейса для отслеживания смешанной реальности в впечатляющие приложения](/windows/mixed-reality/develop/platform-capabilities-and-apis/mixed-reality-capture-for-developers#integrating-mrc-functionality-from-within-your-app)** . разработчики могут интегрировать в свои приложения запись смешанной реальности с помощью встроенного [пользовательского интерфейса Windows для записи данных на камере](/windows/uwp/audio-video-camera/capture-photos-and-video-with-cameracaptureui) , выполнив всего несколько строк кода.

## <a name="new-features-for-hololens"></a>Новые функции для HoloLens

обновление Windows 10 за октябрь 2018 г. общедоступна для всех HoloLens клиентов и включает ряд усовершенствований, таких как:

### <a name="for-everyone"></a>Для всех

* **меню "быстрые действия** " — изнутри приложения смешанной реальности можно нажать кнопку Windows, чтобы открыть меню "быстрые действия" с простым доступом к *началу записи видео*, *взять снимки*, *дома*, *изменить громкость* и *Подключение*.
* **запуск и завершение записи видео в меню "пуск" или "быстрые действия"** . если запись видео запускается из меню меню или быстрые действия, вы сможете отключить запись из того же места. (Не забывайте, что это всегда можно сделать с помощью Voice Commands.)
* **Project на устройство с поддержкой Miracast** . Project содержимое HoloLens на соседнее устройство или телевизор/монитор, если используется дисплей или адаптер с поддержкой Miracast.
* **новые уведомления** — просмотр и реагирование на уведомления на HoloLens, как и на пк.  
* **Полезные наложения в впечатляющих приложениях смешанной реальности** . Теперь вы можете видеть наложения, такие как клавиатура, диалоговые окна, средство выбора файлов и т. д., при использовании впечатляющих приложений смешанной реальности.
* **визуальный индикатор изменения громкости** . при использовании кнопок вверх/вниз на HoloLens вы увидите визуальный индикатор уровня громкости на гарнитуре.
* **Новые визуальные элементы для загрузки устройства** — индикатор загрузки был добавлен во время загрузки, чтобы обеспечить визуальную реакцию на загрузку системы.
* **совместный доступ** . Windows поблизости предоставляет доступ к записи с ближайшим Windows устройством.  
* **общий доступ из Microsoft Edge** Microsoft Edge теперь включает кнопку *общего доступа* . 

### <a name="for-developers"></a>Для разработчиков

* **[интеграция пользовательского интерфейса для отслеживания смешанной реальности в впечатляющие приложения](/windows/mixed-reality/develop/platform-capabilities-and-apis/mixed-reality-capture-for-developers#integrating-mrc-functionality-from-within-your-app)** . разработчики могут интегрировать в свои приложения запись смешанной реальности с помощью встроенного [пользовательского интерфейса Windows для записи данных на камере](/windows/uwp/audio-video-camera/capture-photos-and-video-with-cameracaptureui) , выполнив всего несколько строк кода.

### <a name="for-commercial-customers"></a>Для коммерческих клиентов

* **включить подготовку после настройки** . теперь вы можете применить пакет подготовки среды выполнения в любое время с помощью Параметры.
* **назначенный доступ с помощью групп azure ad** . теперь вы можете использовать группы azure ad для настройки Windows назначенного доступа для настройки конфигурации с одним или несколькими приложениями.
* **ПИН-код для входа в систему при переключении профиля с экрана входа** теперь доступен для пользователя "другой пользователь" на экране входа. 
* **чтение сведений об оборудовании устройства с помощью MDM** . ит-администраторы могут просматривать HoloLens по серийному номеру устройства в консоли mdm.
* **задание имени устройства HoloLens с помощью MDM (rename)** — ит-администраторы могут просматривать и переименовывать HoloLens устройства в консоли mdm.

### <a name="for-international-customers"></a>Для международных клиентов

теперь можно использовать HoloLens с локализованным пользовательским интерфейсом для упрощенного китайского или японского языка, включая локализованную клавиатуру, диктовку, преобразование текста в речь (TTS) и голосовые команды.

## <a name="known-issues"></a>Известные проблемы

мы работаем над тем, чтобы предоставить отличный опыт работы с Windows Mixed Reality, но мы по-прежнему отслеживаем некоторые известные проблемы. Если вы нашли других пользователей, отправьте [нам отзыв](/windows/mixed-reality/give-us-feedback).

### <a name="hololens"></a>HoloLens
 
#### <a name="after-update"></a>После обновления
при использовании обновление Windows 10 за октябрь 2018 г. на HoloLens можно столкнуться со следующими проблемами:
* **Приложения могут завершить цикл входа при запуске из уведомления** . Некоторые приложения, которые требуют входа, могут закончиться в бесконечном цикле входа при запуске из уведомления. например, это может произойти после установки приложения Microsoft Корпоративный портал из Microsoft Store и его запуска из уведомления об установке завершения установки.
* **Страница входа в приложение может быть заполнена пустой страницей** . в некоторых случаях, когда отображается запрос на вход в приложение, при завершении работы страница входа не закрывается, а вместо нее отображается пустая (черная) страница. Можно либо закрыть пустую страницу, либо переместить ее, чтобы обнаружить приложение под. например, это может произойти при входе в систему во время регистрации MDM из приложения Параметры.
