# <a name="other-questions"></a>Другие вопросы

## <a name="my-graphics-driver-isnt-supported-im-getting-graphics-driver-failure-errors"></a>Драйвер графики не поддерживается (я получаю ошибки, возникающие при работе драйвера графики).

Выполните поиск и выполните команду DXDiag:

1.  Если результатом является "базовый модуль подготовки отчетов", графический драйвер не устанавливается. Чтобы устранить эту проблему, выполните следующие действия:
    * Перейдите в **Device Manager > действие > проверить изменения оборудования**.
    * Для обновления драйвера используйте Центр обновления Windows.
    * Если проблема не устранена, перейдите на веб-сайт изготовителя и установите Последнее обновление драйвера. 
    * Если обновление недоступно для GPU, ВМР может не поддерживаться на вашем устройстве. Если вы считаете, что это должно быть, обратитесь в [службу поддержки](https://support.microsoft.com).
2.  Если вы получаете "WDDM 2,1" или более раннюю версию, драйвер графики устанавливается, но может быть не последней версией. Чтобы получить последнюю версию, выполните следующие действия.
    * Для обновления драйвера используйте Центр обновления Windows.
    * Если это обновление не устраняет проблему, перейдите на веб-сайт изготовителя и установите Последнее обновление драйвера. 
    * Если обновление недоступно для GPU, ВМР может не поддерживаться на вашем устройстве. Если вы считаете, что это должно быть, обратитесь в [службу поддержки](https://support.microsoft.com).

Если программа установки Windows Mixed Reality сообщает, что ваша графическая плата не соответствует требованиям и вы считаете, что она работает, убедитесь, что гарнитура подключена к нужной карте.

## <a name="my-samsung-odyssey-or-odyssey-headset-firmware-update-is-stuck"></a>Обновление встроенного по Одиссэй для Samsung или Одиссэй + головной телефон.

Samsung владеет и публикует обновления встроенного по для наушников, поставляемые с помощью приложений для устройств "Samsung ХМД Одиссэй Setup" и "Samsung ХМД Одиссэй + Setup". Для получения дополнительных сведений и получения справки по проблемам с обновлением встроенного по Samsung обратитесь в службу поддержки клиентов Samsung.

Если процесс обновления встроенного по зависает и выполнение не выполнялось более пяти минут:

* Временно отключите все остальные USB-устройства и повторите обновление встроенного по.
* Подключите гарнитуру Samsung к другому порту USB 3,0 на компьютере.
* Отключение или удаление любого установленного программного обеспечения, которое может мешать обновлениям встроенного по, например АОРУС центра приложений Gigabyte.
* Используйте другой ПК для обновления встроенного по гарнитуры Samsung.

## <a name="how-do-i-access-my-pc-desktop-in-mixed-reality"></a>Разделы справки доступ к рабочему столу компьютера в смешанной реальности?
Запустите классическое приложение в гарнитуре с помощью **кнопки Windows > все приложения > Рабочий стол** для доступа к рабочему столу ПК в смешанной реальности.

## <a name="how-can-i-see-multiple-monitors-in-mixed-reality"></a>Как можно увидеть несколько мониторов в смешанной реальности

По умолчанию классическое приложение автоматически переключается для отображения монитора с фокусом. Если вы хотите просмотреть все мониторы в смешанной реальности, сделайте следующее:

* Щелкните значок монитор в левом верхнем углу приложения.
* Отключить параметр "автоматически переключать монитор".
* Выберите монитор, который требуется просмотреть.
* Запустите другой экземпляр классического приложения.
* Выберите монитор, который вы хотите увидеть на этом экземпляре.
* Повторите эти действия для всех физических мониторов.
При каждом перезапуске смешанной реальности необходимо повторно выбрать монитор для отображения в каждом классическом приложении.

## <a name="my-desktop-app-only-shows-a-black-screen"></a>В моем классическом приложении отображается черный экран

Если на компьютере установлен гибридный графический процессор NVIDIA, устройство Nvidia с runtimebroker.exe на дискретном GPU, а не на интегрированном, может быть причиной. Чтобы устранить эту проблему, выполните следующие действия в разделе "[разделы справки создать параметры Optimus для новой программы".](http://nvidia.custhelp.com/app/answers/detail/a_id/2615/~/how-do-i-customize-optimus-profiles-and-settings%3F) чтобы добавить C:\windows\system32\runtimebroker.exe и принудительно запустить его на процессоре с интегрированной графикой. 

## <a name="my-wi-fi-slows-down-when-im-using-windows-mixed-reality"></a>При использовании Windows Mixed Reality Wi-Fi замедляет работу.

Если вы используете подключение Wi-Fi 2,4 ГГц, контроллеры движения могут замедлить работу Wi-Fi:

* Переключитесь на подключение Wi-Fi с частотой 5 ГГц, если оно доступно. [Подробнее](https://support.microsoft.com/help/4000461).
* Используйте отдельный адаптер Bluetooth для подключения контроллеров Motion к компьютеру. См. [Рекомендуемые адаптеры](https://support.microsoft.com/help/4039260/windows-10-mixed-reality-pc-hardware-guidelines).

## <a name="i-got-a-message-that-said-to-plug-in-and-charge-my-pc-why"></a>Я получил сообщение о том, что мы подключимся к своему компьютеру. Почему?

Если вы используете портативный компьютер, Windows Mixed Reality лучше работает, когда ПК полностью заряжен и подключен к сети.

## <a name="what-is-the-experience-options-setting"></a>Что такое параметры работы?

**Параметры > смешанной реальности > отображение гарнитуры > параметры работы** позволяют изменять параметры производительности Windows Mixed Reality. Это позволяет выбрать оптимальную среду для конфигурации оборудования в диапазоне содержимого. Вы можете выбрать один из трех вариантов:
* Автоматически. Windows Mixed Reality определит оптимальное взаимодействие с конфигурацией оборудования. Для большинства пользователей это лучший вариант для начала.
* 60 Гц: Устанавливает частоту обновления 60 Гц и отключает определенные функции, такие как запись видео и предварительный просмотр на портале смешанной реальности.
* 90 Гц: задает частоту обновления 90 Гц.

## <a name="what-languages-are-supported-in-windows-mixed-reality"></a>Какие языки поддерживаются в Windows Mixed Reality

Windows Mixed Reality доступна на следующих языках:

* Китайский (упрощенное письмо, Китай)
* Английский (Австралия)
* Английский (Канада)
* Английский (Великобритания)
* Английский (США)
* Французский (Канада)
* Французский (Франция)
* Немецкий (Германия)
* Итальянский (Италия)
* Японский (Япония)
* Испанский (Мексика)
* Испанский (Испания)

Вы можете использовать Windows Mixed Reality, если ваш компьютер настроен на другой язык. Однако интерфейс будет отображаться в английской (США), а речевые команды и Диктовка будут недоступны. Экранная клавиатура Windows Mixed Reality — это английский (США). Чтобы ввести текст на другом языке, используйте физическую клавиатуру, подключенную к компьютеру. Можно также использовать диктовку в одном из поддерживаемых языков Windows Mixed Reality, перечисленных выше, просто выберите микрофон на экранной клавиатуре.

Windows Mixed Reality также доступна на следующих языках без голосовых команд или функций диктовки:
* Китайская традиционная (Тайвань и Гонконг)
* Нидерландский (Нидерланды)
* корейский (Корея)
* Русский (Россия)

## <a name="how-do-i-uninstall-windows-mixed-reality"></a>Разделы справки удалить Windows Mixed Reality?

1. Отключите гарнитуру от своего компьютера.
2. Закройте портал Windows Mixed Reality.
2. Последовательно выберите пункты  **пуск > параметры > Смешанная реальность** и нажмите кнопку "Удалить".

Когда вы будете готовы приступить к использованию вашей гарнитуры, подключите ее к, и портал Windows Mixed Reality предложит вам выполнить установку.

## <a name="i-got-a-we-couldnt-finish-uninstalling-windows-mixed-reality-message"></a>Я получил сообщение «не удалось завершить удаление Windows Mixed Reality».
