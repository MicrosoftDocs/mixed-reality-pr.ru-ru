# <a name="using-steamvr-with-windows-mixed-reality"></a>Использование Стеамвр с Windows Mixed Reality

Windows Mixed Reality для стеамвр позволяет пользователям запускать стеамвр на Windows Mixed Reality иммерсивное гарнитуры. после установки Windows Mixed Reality для стеамвр пользователи могут запускать свои избранные приложения стеамвр из библиотеки настольных систем или Steam и играть их непосредственно на своей гарнитуре Windows.

## <a name="get-your-pc-ready"></a>Подготовка компьютера

* убедитесь, что отсутствуют ожидающие обновления: выберите **пуск > Параметры > обновление & безопасность > Центр обновления Windows**. Если обновления доступны, выберите **установить сейчас**. Если обновления недоступны, выберите **проверить обновления**, а затем установите новые.
* Требования к компьютеру отличаются для приложений и содержимого в Steam. Ознакомьтесь с минимальными требованиями на заголовок. ПК с графической картой ГТКС 1070 (или аналогичной) и процессором Intel® Core™ i7 должны иметь хорошее представление о широком спектре заголовков.
* настройте [Windows Mixed Reality](set-up-windows-mixed-reality.md) , если вы еще этого не сделали. 

## <a name="set-up-windows-mixed-reality-for-steamvr"></a>настройка Windows Mixed Reality для стеамвр

1. [Скачайте и установите Стеамвр.](https://steamcdn-a.akamaihd.net/client/installer/SteamWindowsMRInstaller.exe)
2. Когда все будет готово, запустите Стеамвр. Руководство по Стеамвр должно запускаться автоматически.

> **Примечание.** Для расширенного устранения неполадок, связанных с установкой Стеамвр, убедитесь, что установлены следующие компоненты программного обеспечения:
> 1. Установите [Steam](http://store.steampowered.com/about/) и **Войдите** или **Создайте новую учетную запись.**
> 2. Установите [стеамвр](https://store.steampowered.com/app/250820/SteamVR/). После подключения гарнитуры Запустите Steam и появится диалоговое окно с предложением установить Стеамвр. Следуйте инструкциям в диалоговом окне, чтобы установить его.
    * Если всплывающее окно не отображается, установите Стеамвр, перейдя к разделу *Tools* *библиотеки*. Найдите Стеамвр в списке и щелкните правой кнопкой мыши и выберите *Install Game (установить игру*).
> 3. установите [Windows Mixed Reality для стеамвр](https://store.steampowered.com/app/719950/Windows_Mixed_Reality_for_SteamVR/).

## <a name="set-up-windows-mixed-reality-for-steamvr-in-an-environment-without-internet-access"></a>настройка Windows Mixed Reality для стеамвр в среде без доступа к интернету

**Хранение необходимых носителей на портативном запоминающем устройстве**
1. установите [стеамвр](https://store.steampowered.com/app/250820/SteamVR/) и [Windows Mixed Reality для стеамвр](https://store.steampowered.com/app/719950/Windows_Mixed_Reality_for_SteamVR/) , как описано выше, используя [Steam](http://store.steampowered.com/about/) на пк с полным доступом к интернету.
2. В Steam откройте раздел Library и найдите часть с меткой "Инструменты".
3. После установки Стеамвр щелкните правой кнопкой мыши запись "Стеамвр" и в появившемся всплывающем меню щелкните запись "Свойства".
4. Откроется новое окно с несколькими вкладками. Выберите вкладку "Локальные файлы" и нажмите кнопку "Обзор ЛОКАЛЬНЫХ файлов".
5. Откроется каталог, содержащий среду выполнения Стеамвр. Скопируйте весь каталог (с именем Стеамвр) на портативный носитель по своему усмотрению (например, флэш-накопитель USB).
6. сделайте то же самое с Windows Mixed Reality для стеамвр и любых приложений, совместимых с стеамвр, которые вы хотите установить на целевом компьютере.

**Запуск Стеамвр на целевом компьютере**
1. После подключения портативного устройства хранения к целевому компьютеру переместите Стеамвр, Микседреалитиврдривер и другие папки в удобное место на целевом компьютере.
![стеамвр и Windows Mixed Reality для стеамвр, установленного на целевом компьютере](images/steamvr-install-files.png)

2. Убедившись, что Стеамвр и Микседреалитиврдривер находятся в одной папке, откройте командную строку. В этом примере предполагается, что содержащаяся папка находится на *к:\стеамвринсталл*. В этом случае в командной строке следует выполнить следующую команду:
```powershell
chdir "C:\SteamVRInstall"
.\SteamVR\bin\win64\vrpathreg.exe adddriver "C:\SteamVRInstall\MixedRealityVRDriver"
```
(обратите внимание, что если вы используете 32-разрядную версию Windows, `win64` часть пути выше должна быть не ниже `win32` .)

это позволит среде выполнения найти Windows Mixed Reality для драйвера стеамвр в пользовательской установке.

4. Чтобы запустить Стеамвр, дважды щелкните файл "vrstartup.exe", расположенный в *SteamVR\bin\win64\vrstartup.exe*, или *SteamVR\bin\win32\vrstartup.exe* , если на целевом компьютере установлена 32-разрядная версия Windows.

[Дополнительные сведения и устранение неполадок см. на странице документации по стеамворкс](https://partner.steamgames.com/doc/features/steamvr/enterprise#2).

## <a name="play-steamvr-games"></a>Играть в игры Стеамвр

1. Подключение гарнитуру на пк и включить контроллеры движения.
2. после загрузки Windows Mixed Reality домашней страницы и отображения контроллеров откройте приложение Steam на рабочем столе.
3. Используйте приложение Steam для запуска игры Стеамвр из библиотеки Steam.

**Совет**. чтобы запускать игры стеамвр без использования гарнитуры, используйте классическое приложение (**запустите > Desktop**) для просмотра рабочего стола ПК и взаимодействия с ним в Windows Mixed Reality.

## <a name="using-motion-controllers-with-steamvr"></a>Использование контроллеров Motion с Стеамвр

В разных играх используются разные контроллеры движения. Вот несколько основ, которые помогут вам приступить к работе:

* Чтобы открыть панель мониторинга Steam, нажмите на левый или правый аналоговый стик прямо вниз.
* чтобы выйти из игры стеамвр и вернуться к Windows Mixed Reality домашней странице, нажмите кнопку Windows.

## <a name="changing-the-resolution"></a>Изменение разрешения

можно настроить ползунок разрешения приложения в окне приложения стеамвр-> Параметры-> в любое время, если вы хотите играть в игры с более высоким разрешением. * * При использовании множителя более высокого разрешения вы можете приостановить работу игры на компьютере. Если увеличить множитель и увидеть снижение производительности, перенастройте ползунок до уровня по умолчанию и перезапустите игру, чтобы убедиться, что изменение вступает в силу.![Настройка разрешения приложения](images/SteamVR_Settings_Applications.png)

## <a name="using-multiple-headsets"></a>Использование нескольких головных телефонов

Если вы работаете с VR, вы можете регулярно использовать на одном компьютере несколько головных телефонов VR. в этом случае обратите внимание, что при подключении Windows Mixed Realityной гарнитуры стеамвр игры всегда будут запускаться на гарнитуре Windows Mixed Reality. если вы хотите запустить игры стеамвр на другой гарнитуре, сначала отключите головную гарнитуру Windows Mixed Reality перед тем, как продолжить.

## <a name="enabling-motion-reprojection-for-steamvr-apps"></a>Включение РЕПРОЕКЦИИ движения для приложений Стеамвр

Windows Mixed Reality для стеамвр — экспериментальная функция репроекции движения, позволяющая повысить уровень перепроекции в 90 кадров/с.

при включении репроекции движения все игры Steam VR будут вырисовываться номинальным по частоте кадров 1/2 (45 кадров/с, а не 90 кадров/с), в то время как Windows Mixed Reality для стеамвр использует векторы движения, созданные графическим процессором для экстраполяции следующего кадра. Для игр Стеамвр, которые надежно наблюдаются в 60/сек + на определенном ПК, это должно привести к заданному беспроблемному интерфейсу 90 в течение всего времени с неудобными артефактами, обеспечивая удобство работы.

Доступны следующие режимы РЕПРОЕКЦИИ движения:

* **параметр стеамвр для каждого приложения**: позволяет управлять репроекцией движения с помощью пользовательского интерфейса стеамвр Параметры. затем можно открыть стеамвр Параметры, перейти в видео > Per-Application видео Параметры и выбрать вариант "сглаживание движения".
* **Auto**: включение РЕПРОЕКЦИИ движения для автоматического включения в случае слишком медленной визуализации игры, чтобы поддерживать 90 кадров/с. Когда игра начинает поддерживать 90 кадров/с или начинает отрисовку меньше 45 кадров/с, переработка движения отключается. Асинхронное ревыполнение РЕПРОЕКЦИИ всегда включено.
* **Вектор движения**: заставляет приложение всегда выполняться в половина кадров с репроекцией вектора движения.
* **Нет**: отключение РЕПРОЕКЦИИ движения.

**Ожидаемый визуальный Artifacts** 

1. При использовании разрешения приложения, превышающего 150%, может возникнуть размытие. При использовании РЕПРОЕКЦИИ движения рекомендуется использовать значение менее 150%.
2. Резкие контрастные края или текст, особенно для Худс в игре или меню, могут выглядеть временно искривленными или искаженными из-за дисокклусион.
3. Стеамвр дома и во многих других играх, на которых не установлено надежное попадание 50-60 кадров/с на ПК, будет по-прежнему работать в этом режиме.
4. Некоторые игры были отправлены на работу с частотой 50% скорости или с увеличением задержки (запаздывание). сообщите об этих играх с помощью [Центр отзывов о Windows](filing-feedback.md) приведенных ниже инструкций.

Изначально у нас есть экспериментальная поддержка современных видеопроцессоров NVidia. Мы продолжаем выполнять итерацию и улучшать поддержку РЕПРОЕКЦИИ движения на других графических процессорах, и мы рады услышать ваши отзывы.

**Поддерживаемые GPU:** Nvidia GeForce GTX1060, AMD RX470 или выше, с установленными Windows Mixed Reality совместимыми графическими драйверами.

Чтобы включить репроекцию движения, сделайте следующее:

1. убедитесь, что вы выбрали **Windows Mixed Reality для бета-версии стеамвр** , выполнив приведенные выше инструкции.
2. Откройте панель мониторинга Стеамвр.
3. нажмите на левой стороне кнопку с логотипом Windows Mixed Reality, чтобы открыть **Windows Mixed Reality для стеамвр** Параметры.
4. В открывшемся пользовательском интерфейсе перейдите на вкладку графика.
5. Выберите "Авто" в качестве значения по умолчанию для режима РЕПРОЕКЦИИ перемещения приложения Стеамвр, чтобы включить автоматическую репроекцию движения.

![Включить индикатор ЛСР & ЛСР с Сеттингсукс](images/settingsux-enable-lsr.gif)

**Индикатор РЕПРОЕКЦИИ движения**

Индикатор РЕПРОЕКЦИИ движения помогает диагностировать проблемы в экспериментальной функции автоматического развертывания движения. Если задано значение true, в левом верхнем углу экрана гарнитуры отображается индикатор во время автоматической реорганизации движения. Цвет и расположение этого индикатора соответствует текущему режиму реконструирования движения. примеры см. на схеме ниже.

![Индикатор mvLS](images/mvLSRIndicator.png)

Зеленый = репроекция движения отключена, так как приложение может быть отображено с полной частотой кадров.

Голубой = репроецирование движения включено, так как приложение привязано к ЦП.

Синий = репроецирование движения включено, так как приложение привязано к GPU.

Red = репроекция движения отключена, так как приложение работает менее половинной частоты; Попробуйте сократить супер выборку, если она включена.

Зеленый + голубой + синий = репроект движения находится в режиме половинной частоты или приложение запросило репроекцию движения.

## <a name="sharing-feedback-on-steamvr"></a>Обмен отзывами по Стеамвр

ваш отзыв не будет ценным, когда дело доходит до улучшения Windows Mixed Reality стеамвр. отправьте все отзывы и ошибки с помощью [Центр отзывов о Windows](filing-feedback.md). Следуйте этим рекомендациям, чтобы помочь нам максимально эффективно использовать ваши отзывы:

1. В центре обратной связи укажите, что вы сообщаете о новой проблеме в разделе «какой тип отзыва?» в верхней части.
2. Выберите категорию **Смешанная реальность** и Подкатегория **приложения** .
3. В сводке проблемы вставьте слово «Стеамвр». Это поможет нам найти ваши отзывы.
4. Опишите, какую игру или приложение Стеамвр вы использовали при работе над проблемой.
5. Попробуйте присоединить отчет Стеамвр System к вашему отзыву. Это предоставляет больше журналов, которые помогут нам диагностировать проблему.
    1. В окне Стеамвр (маленькие окна, в которых отображается состояние контроллера) выберите в заголовке, чтобы открыть меню.
    2. Выберите "создать системный отчет".
    3. Сохранить в файл.
    4. Присоедините созданный файл к записи в центре обратной связи напрямую.
6. Если ваш отзыв относится к Стеамвр производительности, собирайте трассировку производительности смешанной реальности: 
    1. Нажмите кнопку **Повторное создание проблемы** .
    2. В раскрывающемся списке рядом с пунктом "включить данные" выберите " **производительность смешанной реальности**".
    3. Убедитесь, что игра запущена, и выберите **начать запись**.
    4. Потратьте несколько секунд на воспроизведение игры, чтобы захватить трассировку. Не захватывать трассировку более 10-15 секунд, или она будет слишком большой для отправки.
    5. Выберите **прерывать запись**.
7. После завершения всех остальных полей выберите **Отправить** .

Если у вас есть вопросы или комментарии для совместного использования, вы также можете связаться с нами на нашем [форуме Steam](http://steamcommunity.com/app/719950/discussions/).
