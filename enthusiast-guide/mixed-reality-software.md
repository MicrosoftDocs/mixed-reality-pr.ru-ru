# <a name="mixed-reality-software-overview-and-release-history"></a>Обзор программного обеспечения смешанной реальности и история выпусков

## <a name="introduction-to-mixed-reality-software"></a>Введение в программное обеспечение для смешанной реальности

Windows Mixed Reality состоит из следующих основных программных компонентов:

1. **портал смешанной реальности**, предоставляющий основной интерфейс Windows Mixed Reality
    * в Windows 10 версии 1709 и 1803 портал Mixed Reality — это ключевой компонент Windows 10 операционной системы, обновленный с помощью Центр обновления Windows.
    * в Windows 10 версии 1809 и более поздних портал смешанной реальности обновляется с помощью приложения Microsoft Store.
    * в Windows 11 версии 21H2 портал смешанной реальности обновляется с помощью приложения Microsoft Store.
2. **Пакет по запросу "функция смешанной реальности** " (FOD) автоматически загружается и устанавливается во время первого запуска портала смешанной реальности. Дополнительные сведения о пакете FOD можно найти [здесь](/windows/application-management/manage-windows-mixed-reality) .
3. **драйвер наушников и контроллеров движения смешанной реальности**, также известный как драйвер датчиков HoloLens, является пакетом драйверов, который позволяет Windows Mixed Realityным гарнитурам работать с Windows Mixed Reality. он автоматически скачивается и устанавливается с помощью Центр обновления Windows при первом подключении гарнитуры смешанной реальности и регулярно обновляется с помощью Центр обновления Windows
4. Драйверы модели контроллера движения "* * Mixed Reality" содержат трехмерные модели контроллеров движения смешанной реальности, необходимые для работы в смешанной реальности. это автоматически скачивается и устанавливается с помощью Центр обновления Windows при первом подключении контроллеров движения смешанной реальности к компьютеру и обновления с помощью Центр обновления Windows
5. **Windows 10, версия 1709 (обновление создателя отправителя) или более поздней версии** содержит ключевые компоненты ос и технологии, обеспечивающие Windows Mixed Reality

для использования Windows Mixed Reality в стеамвр требуется следующее программное обеспечение:

6. **Стеамвр**, разработанная и обслуживаемая корпорацией вентилей, которая позволяет приложениям и играм Virtual Reality на Steam. Дополнительные сведения можно найти [здесь](https://go.microsoft.com/fwlink/?linkid=862788)
7. **Windows Mixed Reality для компонента стеамвр** , который связывает стеамвр с Windows Mixed Reality. дополнительные сведения об этом компоненте можно найти [на странице Windows Mixed Reality для стеамвр](http://store.steampowered.com/app/719950/Windows_Mixed_Reality_for_SteamVR/) .

управление гарнитурой Windows Mixed Reality:

8. **приложение-помощник по устройствам**, разработанное и обслуживаемое каждым из производителей гарнитуры, содержит краткие сведения о гарнитуре Windows Mixed Reality. на гарнитурах со встроенными возможностями Bluetooth приложение "помощник по устройствам" позволяет восстанавливать контроллеры движения в фабрику Bluetooth связывания. Некоторые гарнитуры (например, Samsung Одиссэй и Samsung Одиссэй +) также используют приложение "помощник по устройствам" для предоставления обновлений встроенного по в гарнитуру от производителя гарнитуры. это приложение скачивается автоматически при первом подключении гарнитуры и может быть найдено в меню "пуск" Windows.

## <a name="windows-11-release-notes---october-2021"></a>заметки о выпуске Windows 11 — октябрь 2021

### <a name="infinite-expanse"></a>Бесконечное заданной

<img src="images\infinite-expanse-win11.png" alt="The Infinite Explanse environment">

<br>

* новая виртуальная домашняя среда для Windows Mixed Reality устройств с значительной экономией области и размера, оптимизирована до отдельных этапов, а не более широких клиффхаусе. 
* При разработке с учетом производительности бесконечные заданной были разработаны для решения долгосрочных запросов клиентов с менее ресурсоемкими виртуальными домашними средами, что позволяет клиентам максимально эффективно работать с играми и опытом. 
* Эту новую среду виртуальной домашней среды можно найти на **панели контакты** в меню **места** . 

### <a name="steamvr-boot-with-mixed-reality-portal-launch"></a>Загрузка Стеамвр с запуском портала смешанной реальности

* Новый параметр, доступный для автоматического запуска Стеамвр при запуске ВМР, позволяющий обходить ВМР Home Space и переходить непосредственно к Стеамвр.
   * этот новый параметр можно найти в **Параметры** приложении в разделе **смешанная реальность > запуск и настольная > автоматический запуск**.
    
### <a name="new-startup-experience-settings"></a>Новые параметры интерфейса запуска

* Доступны новые параметры для лучшей настройки идеального запуска, повышая уровень контроля над запуском портала смешанной реальности.
* Теперь можно управлять запуском портала смешанной реальности при подключении устройства или при активации датчика присутствия, а также управлять открытием приложения виртуального рабочего стола.
* эти новые параметры можно найти в **Параметры** приложении в разделе **смешанная реальность > запуск и рабочий стол** .
    * Переключитесь, чтобы запустить модуль MRP для подключаемого модуля ХМД.
    * Переключите, чтобы запустить MRP при обнаружении присутствия.
    * Переключитесь на режим открыть классическое приложение в классическом приложении.
